# 대본 신규 주장 팩트체크

## 역할

대본 검수에서 추출된 신규 주장(verified-data.md에 없는 비교/수치/사례)을 웹 검색으로 검증한다.

## 입력

- 신규 주장 목록 (review.md의 신규 주장 검수 섹션에서 추출)
- verified-data.md (기존 검증 데이터, 참고용)

## 작업

### 1단계: 주장별 검증

각 신규 주장을 WebSearch로 검증한다:
- 비교 주장: 양쪽 수치/조건이 현실과 맞는지 확인
  - 예: "170만원으로 3인 가족 동남아 일주일" → "3인 가족 동남아 여행 비용" 검색
- 수치 주장: 해당 수치의 정확성 확인
- 사례 주장: 해당 사건/사례의 사실 여부 확인

검증 결과 3단계 분류:
- ✅ 검증됨: 현실과 부합
- ❌ 불일치: 현실과 다름
- ⚠️ 과장: 방향은 맞지만 정도가 과장됨

### 2단계: draft-factcheck.md 작성

| # | 주장 | 검증 | 실제 | 수정제안 | 출처 |
|---|------|------|------|----------|------|
| 1 | ... | ✅/❌/⚠️ | ... | (❌/⚠️일 때만) | ... |

## 주의사항

- 검색어는 주장의 핵심 조건을 모두 포함 (인원수, 금액, 기간 등)
- "대략 맞다"가 아니라 구체적 조건에서 맞는지 확인 (일주일 vs 3~4일 등)
- 수정제안은 실제 검색 결과 기반. 추측 금지.
