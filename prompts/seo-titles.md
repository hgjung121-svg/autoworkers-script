# SEO 제목 SSOT — A/B/C 3패턴 공식

`news-latest-3` 및 `script-pd` 양쪽 공유. 모든 영상은 제목 3안을 제시하고 사용자가 선택 ([feedback_title_options]).

---

## 3패턴 공식

### 패턴 A — 숫자+위기+행동형 (CTR 최강)

**구조**:
```
[숫자] + [감정/위기 키워드] + [핵심 키워드] + [행동 유도]
```

**예시**:
- "충격! 12월 대출금리 또 오릅니다, 지금 당장 확인할 3가지"
- "한국은행 발표! 내년 물가 이렇게 됩니다 (시니어 필수 시청)"
- "60대 이상 자산 1억 사라진다, 오늘 안에 점검할 5가지"

**강점**: 클릭률 최강. 긴급성 + 행동 유도.
**약점**: 모든 영상에 쓰면 시청자 피로감 누적.
**적합**: 정책 변화 영향 / 시장 폭락·급등 / 마감 임박 신청

---

### 패턴 B — 질문+공감형 (체류시간 ↑)

**구조**:
```
[질문] + [시니어/타겟 페인포인트] + [해결 약속]
```

**예시**:
- "60대 이상 꼭 보세요, 연금 깎이는 이유와 대책 5가지"
- "전세금 못 받을 위기? 정부가 오늘 내놓은 해결책 전격 분석"
- "은퇴 후 통장 비는 진짜 이유, 이거 모르면 평생 손해"

**강점**: 체류시간 길어짐. 시청자 자기 문제로 인식.
**약점**: 클릭률은 A보다 약함. 호기심 자극이 핵심.
**적합**: 노후·연금 / 부동산 페인포인트 / 의료비·세금

---

### 패턴 C — 속보+권위형 (속보 모드 전용)

**구조**:
```
[속보 키워드] + [발표 주체] + [핵심 수치] + [영향]
```

**예시**:
- "[속보] 한국은행 금리 0.25%p 인상, 우리 대출이자 얼마 오르나"
- "[방금발표] 기재부 부동산 대책, 1주택자 반드시 확인하세요"
- "[긴급] 미국 Fed 금리 동결, 한국 주식·환율 영향 분석"

**강점**: 신뢰 + 속도. 알고리즘 트렌딩 진입 가능.
**약점**: 속보가 아닐 때 쓰면 클릭베이트 신호.
**적합**: 속보 모드 전용 / 24시간 내 공식 발표 영상

## 제목 작성 필수 룰

### 1. 글자 수
- **앞 30자 안에 핵심 키워드 배치** — 모바일 잘림 대응
- **전체 50~60자** (한글 기준)
- 따옴표 형식이면 60자까지 허용 (예: `"~라고요?" + 본제목`)

### 2. 숫자 포함
- 가급적 모든 제목에 숫자 1개 이상
- 형식: "3가지", "5%", "12월", "60대"
- 숫자가 없으면 약속이 추상화되어 CTR 하락

### 3. 감정 단어
- 충격, 비상, 긴급, 주의, 꼭, 반드시, 절대
- ⚠️ 과장 금지 — 본문에서 입증 못 하면 클릭베이트
- 거짓말 NO ("당신은 모릅니다" 류 ❌)

### 4. 대괄호 활용
- `[속보]`, `[긴급]`, `[발표]`, `[심층분석]`, `[방금발표]`
- 클릭률 +15~30% 효과
- 속보 모드 전용 사용 권장 (남용 시 가치 하락)

### 5. 부등호 사용 금지 ([feedback_youtube_no_angle_brackets])
- ❌ `<1%` → ✅ "1% 미만"
- YouTube Studio가 차단함

### 6. AI 패턴 회피 (13게이트 게이트 1)
- ❌ "알아보겠습니다", "살펴봅시다", "정리해드립니다"
- ✅ 구체 약속 + 결과 형태

## 3안 제시 포맷

영상 완료 후 `output/youtube.md`의 `## 제목 후보`에 다음 형식으로 기록:

```markdown
## 제목

{확정 제목 — 사용자가 선택한 1개}

## 제목 후보

1. **패턴 A** — "충격! 한국은행 0.25%p 인상, 12월 대출이자 즉시 확인할 3가지"
   - [강점] 긴급성+숫자+행동 유도 → 클릭률 ↑
   - [약점] 시청자 피로감 가능

2. **패턴 B** — "60대 이상 꼭 보세요, 한국은행 금리 인상이 노후 자금에 미치는 진짜 영향"
   - [강점] 시니어 페인포인트 직격 → 체류시간 ↑
   - [약점] 클릭률은 A보다 약함

3. **패턴 C** — "[속보] 한국은행 0.25%p 인상 결정, 가계 대출에 어떤 영향?"
   - [강점] 권위+속도 → 트렌딩 진입 가능
   - [약점] 속보 모드 아니면 부적합
```

## 제목 → 구조 제약 룰 (메모리 [feedback_youtube_md_quality])

확정 제목이 본문 구조를 제약. outline.md 생성 시 반영:

| 제목 패턴 | 구조 제약 |
|---|---|
| "N가지 이유/방법/실수" | 정확히 N개 섹션 |
| "~의 몰락/추락/붕괴" | 시간순 서사 |
| "A vs B" 비교 | 비교 프레임워크 |
| "진짜 이유/숨겨진 비밀" | 미스터리 구조 (표면→단서→진실) |
| "~하는 법" | 프로세스 구조 |
| "[속보] ~ 영향" | 사건 → 영향 → 대응 3단 구조 |

## 검증 — Title ABCD

작성한 제목 3안에 대해 자가 점검:
- **A**ction — 클릭 후 시청자가 얻을 행동/결과가 명확한가?
- **B**enefit — 시청자에게 어떤 이득(정보·돈·시간)을 약속하는가?
- **C**redibility — 신뢰감 있는 어조인가? 과장·거짓 없는가?
- **D**iscoverability — SEO 키워드 2~3개 포함, 검색 가능한가?

4개 다 ✅ 아니면 재작성.
