# TTS 대본 전처리 규칙

⚠ 절대 원칙: 원본 내용(문장, 단어 선택, 어순, 의미)은 절대 변경하지 않는다. 형식만 정리한다.
- 문장을 다시 쓰거나, 표현을 바꾸거나, 요약하거나, 순서를 바꾸지 않는다.
- 아래 규칙에 해당하는 형식 요소만 제거/수정한다.

## 정리 규칙

1. 나레이션 텍스트만 출력 (제목, 장면번호, 연출지시 등 제거)
2. 문장이 끝나면 온점(.) 뒤에 공백 1칸
3. 따옴표("") 사용 최소화 — 직접 인용이 아니면 제거
4. 문단 구분은 빈 줄(줄바꿈)로
5. 특수문자(※ ★ → 등), 이모지 사용 금지
6. URL, 이메일 주소 삽입 금지
7. 단어 뒤 괄호 설명 금지 (예: 클로드(Claude) → 클로드)
