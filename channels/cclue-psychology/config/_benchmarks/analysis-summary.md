# 벤치마크 분석 요약

## 분석 영상
1. 천재의 두뇌 만들 수 있습니다 (뇌과학적 공부법) — 674,871 조회수, 한눈에 심리학 (18분)
2. 가장 가깝다고 생각한 사람들이 내 불행을 즐기는 심리 — 166,769 조회수, 일상심리학 (23분)
3. 펜타닐 한번만 해도 인생 나락 가는 이유 — 4,352,456 조회수, Jisik (5분)

## 공통 패턴
- **톤**: 존댓말. TED 강연체 또는 감성적 문학체. 청자에게 직접 질문으로 시작
- **서사 구조**: 공감 유도 → 원인/심리 분석 → 해법/통찰
- **훅 방식**: 청자의 공통 경험 소환 ("시험 전날 밤새웠는데 하나도 생각 안 났던 경험")
- **전달 방식**: 심리/뇌과학 데이터 + 일상 비유 병합
- **평균 영상 길이**: 5~23분 (포맷 다양)

## 썸네일 패턴 (레퍼런스 3개 분석)
- 스타일: 일러스트/애니메이션 강하게 선호 (실사 없음)
- 캐릭터: 얼굴 없는 단순 캐릭터, 스틱맨, 3D blob 캐릭터
- 색감: 파스텔/소프트 또는 심플 모노톤
- 텍스트: 굵고 큰 키워드 중심, 2줄 구조
