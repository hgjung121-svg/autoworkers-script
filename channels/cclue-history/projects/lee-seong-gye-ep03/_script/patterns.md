## 1. 반복되는 훅 공식

### 관찰

**훅 원문 비교**:
- 영상 001 (350K): "예사롭지 않은 눈빛의 남자" — 정도전을 감각적으로 묘사하며 Show Don't Tell형 훅. 인물의 등장 자체를 미스터리화.
- 영상 002 (181K): "누구도 생각지 못한 일" + 불 지르는 재현 장면 — 패턴 인터럽트(시각적 충격) + Show Don't Tell 조합. 결과를 먼저 보여주고 과정을 궁금하게 만듦.
- 영상 003 (131K): "최정태세문단세" 교과서 암기법 소환 — 권위형 + 공감형. 학창 시절 공유 경험으로 즉각 동일시.

**공통 패턴**:
1. **인물/사건의 감각적 제시**: 3편 모두 첫 문장에서 추상적 설명 대신 구체적 장면·이미지·감각을 제시한다.
2. **궁금증 설치**: 결과나 분위기를 먼저 보여주되 "왜?"를 남겨둔다 (001: 예사롭지 않은 눈빛 → 누구인가, 002: 불 지르기 → 왜 태웠나, 003: 왕 순서 → 첫 왕의 비밀).
3. **Show Don't Tell 우위**: 고퍼포먼스 영상일수록 설명 대신 장면 재현으로 훅을 건다.

**인트로 공통 역할**:
- 001: 시리즈 전체 방향성 제시 + 두 인물의 운명적 만남 예고
- 002: 퀴즈형("조선 건국 후 제일 먼저 한 일이 뭐겠습니까?") + 백성 편임을 증명하며 감정 이입 확보
- 003: "영화·드라마에서도 다 담지 못했던 비하인드 스토리"라는 독점 정보 약속

공통점: **시청 혜택을 구체적으로 약속**한다. "배신과 복수의 조선 건국기", "왜 역적으로 몰렸는지", "비하인드 스토리" 등 이 영상에서만 얻을 수 있는 것을 명시.

### 설계 원칙

1. **첫 문장은 장면으로 시작한다**: 설명("오늘은 ~에 대해 알아봅니다")이 아니라, 인물의 행동·표정·공간을 묘사하는 문장으로 연다.
2. **훅에 '왜?'를 심는다**: 결과 또는 장면을 먼저 보여주되, 과정이나 이유는 비워둔다. 고퍼포먼스 영상 001·002가 이 구조를 사용.
3. **인트로 30초 안에 시청 혜택을 명시한다**: "이 영상을 끝까지 보면 ~를 알게 됩니다"를 직접적으로든 암시적으로든 전달한다.
4. **백성/서민 관점의 감정 이입을 초반에 확보한다**: 영상 002가 "백성 편인 인물임을 증명"하며 시청자 공감을 가장 빠르게 확보 — 우리 채널 정체성과 일치.

## 2. 반복되는 본론 구조 & 전달 기법

### 관찰

**구조 비교**:
- 영상 001: 동지 결성 → 내부 갈등 → 외부 위기(요동 정벌) → 회군 결단 → 숙청 → 건국 → 또 다른 갈등의 씨앗
- 영상 002: 개혁(공) → 건국 → 설계(한양·경복궁) → 위기(세자 책봉) → 처형 → 금기화(467년) → 복권·부활
- 영상 003: 낙인 집안 → 반등 → 영웅화 → 혁명가 연대 → 배수진 → 역전

**공통 서사 흐름**:
> **기원/배경 → 상승(영웅화·개혁) → 위기(갈등·대립) → 결단(회군·처형) → 결과(건국·몰락) → 씨앗(다음 갈등 예고)**

3편 모두 "상승 후 추락" 또는 "고난 후 역전"의 반전 구조를 핵심으로 삼는다. 단선적 상승이 아니라 반드시 중간에 위기·갈등 파트가 삽입된다.

**전달 기법**:

*비유*:
- 영상 003: "요즘으로 치면 스캔들" — 현대 일상 비유로 즉각 이해를 돕는 방식. 3편 중 가장 적극적으로 비유 사용.
- 영상 001·002: 드라마적 서사 자체가 비유를 대체 — 장면 묘사로 추상적 개념을 구체화.

*숫자*:
- 영상 003: "왜구 500여 회", "3천 vs 수만" — 대비 쌍으로 숫자를 제시하여 스케일을 체감시킴.
- 영상 002: "467년간 역사에서 지워졌다" — 단일 숫자의 충격적 크기로 감정 유발.

*사례*:
- 영상 003: 까마귀 5마리 실록 기록, 아지발도 투구 끈 저격 — 미시적 디테일로 현장감.
- 영상 002: 토지 문서 소각 장면 — 구체적 행위로 개혁의 파괴력 체감.
- 영상 001: 위화도 고립 장면 — 공간적 고립감으로 긴장 극대화.

**리텐션 장치**:
- **파트 전환 시 오픈루프**: 3편 공통. 영상 003: "정도전이 왜 이성계를 선택했을까요 바로 그 비밀을", 영상 001: 배신 구도 전환 시마다 오픈루프 설치.
- **중반 패턴 인터럽트**: 영상 003: 까마귀 5마리 실록 기록 (예상 밖 디테일), 영상 002: 토지 문서 소각 장면 (시각적 충격).
- **결말 클리프행어**: 영상 001: 세자 책봉 파동, 영상 003: "다음 시간에 이어가겠습니다" — 시리즈 연속성.

### 설계 원칙

1. **본론 뼈대는 "상승 → 위기 → 결단 → 결과" 4박자로 잡는다**: 단순 연대기 나열이 아니라, 반드시 중간에 갈등/위기 파트를 배치하여 긴장을 만든다.
2. **비유는 추상적 개념이 나올 때마다 배치한다**: 특히 정치적 판단, 권력 역학 등 시청자가 체감하기 어려운 부분에 현대 일상 비유를 삽입한다.
3. **숫자는 반드시 대비 쌍 또는 충격적 단일 수치로 제시한다**: "왜구 500여 회"처럼 스케일을 체감시키거나, "467년" 같은 단일 숫자의 충격을 활용한다.
4. **파트 전환 직전에 오픈루프를 설치한다**: "그런데 이 결정에는 숨겨진 이유가 있습니다"와 같이 다음 파트의 궁금증을 심는다.
5. **중반에 패턴 인터럽트(예상 밖 디테일)를 1회 이상 배치한다**: 실록의 이색 기록, 현장 묘사 등으로 시청자의 주의를 환기한다.
6. **결말은 반드시 "씨앗"을 남긴다**: 이야기를 완전히 마무리하지 않고, 다음 갈등의 씨앗(이방원의 소외, 정도전의 위기 등)을 심어 시리즈 연속성을 확보한다.

## 3. 감정 증폭 전략

### 관찰

**댓글 감정 분석**:

| 영상 | 조회수 | 1순위 감정 | 2순위 감정 | 3순위 감정 |

- 영상 002 또 다른 고공감 댓글(좋아요 28): "그럼 뭐하나... 결국 백성 입장에선 시간이 흐르면서 권문세족이 또다른 이름의 세력으로 땅을 빼앗긴건 같았는데" — **백성 관점의 허탈함**.
- 영상 003 최다 공감 댓글(좋아요 18): "조선왕도 전쟁사처럼 시리즈로 나오는 건가요?" — 시리즈 기대감.

**가장 뜨거운 감정 종합**: **논쟁을 유발하는 양면적 해석** > **백성 관점의 허탈함/분노** > **시리즈 연속성 기대**

**감정을 만든 장치 역추적**:
- 양면적 해석: 영상 002가 정도전을 "조선의 설계자"로 영웅화하되, 권력욕 측면도 암시 → 시청자가 스스로 판단하려는 욕구 자극.
- 백성 관점의 허탈함: 영상 002에서 토지 개혁의 이상과 현실의 괴리를 보여줌 → 시청자가 자연스럽게 백성 편에 서서 분노.
- 긴장/배신감: 영상 001에서 위화도 고립 장면, 건국 후 이방원 소외 → 동지가 적이 되는 구도가 감정을 강하게 자극.

**감정 흐름 비교**:
- 영상 001: 긴장(초반) → 통쾌함(중반) → 배신감·불안(후반)
- 영상 002: 통쾌함(초반) → 긴장(중반) → 충격·연민(후반) → 허탈함 → 감동(부활)
- 영상 003: 호기심(초반) → 경이감(중반) → 긴장·비극(후반)

공통: **후반부에 부정적 감정(배신감·충격·비극)이 배치**되어 여운을 남긴다. 초·중반의 긍정적 에너지(통쾌·경이)가 후반의 비극과 대비되어 감정 낙차를 만든다.

### 설계 원칙

1. **"양면적 해석의 여지"를 의도적으로 심는다**: 인물을 일방적 영웅/악인으로 그리지 않고, 시청자가 스스로 판단할 수 있는 정보를 제공한다. 이것이 댓글 논쟁(= 참여도)을 유발하는 가장 강력한 장치다.
2. **백성/서민 관점의 허탈함을 반드시 1회 이상 삽입한다**: "결국 백성에게는 뭐가 달라졌는가?"라는 질문을 대본에 직접 넣어, 우리 채널 정체성과 시청자 감정을 동시에 잡는다.
3. **감정 흐름은 "긍정 에너지 → 비극적 낙차 → 여운"으로 설계한다**: 초반 30%에서 영웅화/통쾌함으로 시청자를 끌어들이고, 후반 30%에서 배신/비극으로 감정 낙차를 만들며, 마지막에 여운(씨앗·질문)을 남긴다.
4. **분노 축적 → 통쾌 해소 사이클을 최소 1회 배치한다**: 영상 002의 "권문세족 횡포 → 토지 문서 소각"이 가장 강한 감정 반응을 얻었다.

- 가장 많이 사용된 구도: **인물 대비(A vs B)** — 3편 중 2편(002, 003)이 사용, 001도 2인 초상화 대비 구도.
- 색채 공통점: **어두운 배경 + 노란색 또는 빨간색 포인트** — 3편 모두 다크 톤 위에 강렬한 포인트 색상.
- 고퍼포먼스 영상의 비주얼 특징: 001은 MC의 놀란 표정이 감정 전달의 핵심. 초상화에 감정을 부여하는 장치(002의 말풍선 "억울")가 효과적.
- 모바일 가시성: 3편 모두 인물 얼굴이 크게 배치되어 160x90px에서도 인물 식별 가능. 텍스트는 최대 폰트 요소 기준 식별 가능하나, 소형 텍스트는 모바일에서 읽기 어려움.

**썸네일 비주얼 추가 분석**:
- 스크롤 멈춤 요인: (1) 역사 인물 초상화의 이질감(현대 출연자와의 시대적 대비), (2) 표정에서 오는 감정 신호(놀람, 안타까움), (3) 어두운 배경 위 노란색 텍스트의 높은 가독성.
- 말풍선 장치(002): 무표정한 초상화에 "억울"이라는 감정을 직접 부여 → 역사 인물에 인격을 불어넣어 클릭 유도.

**제목 비교**:
- 001: `[#벌거벗은한국사] 혼란 그 자체였던 고려 말💥 최태성이 들려주는 《태조 이성계와 정도전의 조선 건국기》`
  - 장치: 이모지, 강사 권위, 시리즈 태그
- 002: `고려를 뒤집고 조선을 세운 과감함! 조선의 설계자였으나, 함께 건국했던 이방원에게 죽임 당한 정도전의 비참한 최후`
  - 장치: 대비(설계자 vs 비참한 최후), 구체적 인물 관계, 감정어(비참한)
- 003: `정도전이 이성계를 조선의 왕으로 선택한 이유! 위화도 회군은 필연적이었다?`
  - 장치: 질문형("필연적이었다?"), 권위(교수), 시리즈

**공통 제목 구조**: **행위/사건 + 결과/감정어**. 특히 002(최고 CTR 추정)는 "설계자 → 비참한 최후"의 대비가 가장 극적.

**썸네일-제목 역할 분담**:
- 001: 썸네일 = "배신과 복수"(감정), 제목 = "혼란 그 자체였던 고려 말"(배경 설명)
- 002: 썸네일 = "역적으로 몰려 죽은 일등공신"(비극적 결과), 제목 = "설계자 → 비참한 최후"(과정과 결말)
- 003: 썸네일 = "칼은 왜 고려로 향했나"(질문), 제목 = "선택한 이유! 필연적이었다?"(답 약속)

공통: **썸네일은 감정/결과를 압축**, **제목은 과정/이유를 부연** — 정보 중복 없이 역할을 나눔.

### 설계 원칙

**썸네일 텍스트 설계 원칙**:
1. 텍스트 요소 3~4개, 요소당 7~11자, 총 30~40자 이내로 구성한다.
2. 핵심 키워드 1개는 최대 폰트로 배치하고, 감정 보조 키워드는 소형(괄호/말풍선)으로 분리한다.
3. 반드시 대비 구조를 텍스트에 직접 드러낸다 (영웅 vs 역적, 동지 vs 적, 공신 vs 처형).

**제목 설계 원칙**:
1. "행위/사건 + 결과/감정어"의 대비 구조를 사용한다.
2. 썸네일이 감정/결과를 담당하므로, 제목은 "왜?"와 "과정"에 집중한다.
3. 질문형("~였다?") 또는 감정어("비참한", "혼란") 중 하나를 반드시 포함한다.

**역할 분담 공식**:
- 썸네일 텍스트 = 감정 + 결과 압축 (클릭 충동)
- 제목 = 과정 + 이유 부연 (기대감 구체화)
- 정보 중복 금지. 썸네일에서 말한 것을 제목에서 반복하지 않는다.

**썸네일 비주얼 원칙**:
1. 어두운 배경 + 노란색 또는 빨간색 포인트 텍스트를 기본으로 한다.
2. 인물 대비 구도(A vs B)를 기본으로 사용하되, 인물 얼굴을 크게 배치하여 모바일 가시성을 확보한다.
3. 초상화/인물에 감정을 부여하는 장치(말풍선, 표정, 시선)를 사용한다.
