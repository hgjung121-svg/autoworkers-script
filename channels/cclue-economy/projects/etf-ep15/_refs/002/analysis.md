## 3. 훅 & 인트로 분석

**훅**
> "미국 주식에 투자한다면 모를 수 없는 ETF가 바로 큐큐죠. … 머지 않아 나스닥백 ETF는 QQQ라는 공식이 깨질 가능성이 매우 높아졌습니다."

- 권위형 자존심 인정 + 역설적 선언 + 시청자 행동 직접 부정
- 1차 타겟: **이미 QQQ/QQQM 매수 중인 투자자**

**인트로**
> "QQQ … 운영 규모 5위 … 국내 ETF 시장의 1.5배 …"

- 80조원 규모로 사건성 증명
- CTA는 클로징에만 — 우리 채널 룰과 정반대

## 4. 본문 구조 분석

**포인트 순서**
1. 현 시장: QQQ 80조원, 5위 / QQQM 0.15% / 인베스코 독점
2. 사건: 2026-04-06 블랙록 IQQ 신청 / 04-07 SSGA 신청 (SEC 3개월=6~8월 상장)
3. 네이밍: I = 아이셰어즈(IVV+IQQ 라인업)
4. 본질=보수 경쟁: SPY 0.03 / SPLG 0.02 / SPYG·SCHG 0.04 / QQQ 0.18 / QQQM 0.15
5. 예측: IQQ 보수 0.10~0.12%
6. 역사: SPY→VOO·IVV 추월 (0.09 vs 0.03)
7. 행동 가이드: 보유분 갈아타지 말 것 / 신규는 IQQ / QQQM 적립 중단 X / QQQM 보수 인하 가능성

**구조 패턴**: 현황→사건→의미→본질→예측→역사→행동. **몰락 서사(인베스코 독점 붕괴) + 반전(SPY 전례)** — 우리 채널 선호 서사와 일치.

**전달 기법**
- 데이터: 보수율 6종, 운용규모, 신청일
- 사례: "SPY가 S&P500이었습니다" 동격 강조
- 스토리텔링: 인베스코 독점→블랙록 도전(4/6)→SSGA 가세(4/7)

**리텐션**
- 오픈루프: "QQQM만 바라볼 이유 없다"
- 패턴 인터럽트: "많이 비싼 거예요"
- 1인칭 회상: "저도 옛날 사람이라 스파이를 가져다 쓰거든요"
- 약한 클로징, Q1·Q3 없음

## 5. 감정 트리거

1. **불안(FOMO)** — "더 비싼 비용 지불할 이유 없다"
2. **호기심** — "어떤 모습인지 궁금하신 분들"
3. **통쾌함** — "운용사들은 죽 마시고 우리는 수혜자"

**감정 흐름**: 호기심→불안→안도(SPY→VOO)→통쾌함→행동 안도 — 소비자 관점 100%

**인트로 재확인**: 방향성 일치, 톤 부드럽게 변주

**본문 전달**: ①IQQ 정체 ②상장 시기(6~8월) ③예상 보수(0.10~0.12%) ④행동 가이드 ⑤SPY→VOO 전례
