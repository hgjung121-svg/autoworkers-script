# 패턴 추출 — vidiq 부동산 분석 기반 (4편, 레퍼런스 미수집)

> 카테고리 로테이션: 1·2편(투자/연금)·3편(금리·환율) → 4편 **부동산**으로 전환. vidiq 키워드 광맥 + 폭발 패턴 재활용.

## 1. 후킹 패턴
- 손실회피 공포 > 욕망. "내 전세보증금 떼인다"는 시니어+전월세 직장인 공통 최대 공포.
- 공포+숫자+반전+권위(법조항·HUG·등기부) 인용.

## 2. 훅 유형
- 메인 = **손실회피 공포형(구체 시나리오)** → "전세 만기인데 집주인이 돈이 없대요"
- 보조 = **자가진단형** → "내 보증금도 위험한지 지금 확인"
- 금지: "오늘은~", AI 도입 패턴, 공포 과장·특정 지역/단지 단정.

## 4. 제목 (vidiq CTR 점수 검증 완료)
시드 확정: **"전세 만기인데 집주인이 돈이 없대요, 보증금 떼이기 전 확인할 3가지"** (CTR **84점**)
- 대안: "내 전세보증금 돌려받을 수 있을까, 역전세·깡통전세 자가진단법"(82), "깡통전세 시대, 내 보증금이 위험한지 5분 만에 확인하는 법"(82)
- ★제목이 "3가지" 약속 → 본문 정확히 3개 확인 항목 구조 고정 (제목→구조 제약 룰)★
- 공식: [구체 시나리오] + [공포] + [숫자 3가지]. 공포 과장·지역 단정 0건.

## 5. 핵심약속 (제목→본문 정합)
전세 만기에 보증금 못 돌려받는 상황(역전세·깡통전세)이 왜 생기는지 + 내 보증금이 위험한지 가르는 확인 3가지(전세가율·등기부 권리관계·보증보험) + 위험할 때 대응 순서. 본문 20% 이내 "확인 3가지" 윤곽.

## 6. 리텐션 장치
- 오픈루프: "근데 이거 하나만 빠뜨려도 앞의 확인이 다 무용지물이야" (확인 항목 사이)
- 감정 곡선: 불안(만기인데 돈이 없대) → 이해(왜 이런 일이) → 통제감(3가지로 진단·대응)
- Q1 선언택일: "내 보증금 안전하다고 봐(A) / 불안하다(B)?"

## 7. 안전 주의 (게이트)
- 특정 지역·단지·집주인 단정 금지. "전세사기" 표현은 일반 현상 설명, 특정 지목 X.
- 법률·보증 절차는 일반 정보. "반드시 받는다/무조건 안전" 단정 금지 → "통상·요건 충족 시".
- 부동산 전망(집값 오른다/내린다) 단정 금지. 투자·매수 권유 금지.
