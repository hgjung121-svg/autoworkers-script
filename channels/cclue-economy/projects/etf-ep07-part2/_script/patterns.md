## 1. 반복되는 훅 공식

### 관찰

| 002 돈쭐남 김경필 | 37.7만 | "혼줄이 여러분 안녕하십니까? 돈으로 혼줄내는 남자…돈줄남 김경필입니다" | 브랜드 시그니처형 (약함) |

**공통 훅 패턴 (상위 퍼포먼스 기준)**:
1. **시청자의 누적 피로/갈망을 한 문장으로 언어화** — 004의 "제발 그만 싸우세요"는 "이미 여러 영상을 봤는데 결정 못 했다"는 감정을 대변. 003의 "제2의 월급, 가만히 있어도"는 직장인의 배당 로망을 그대로 호출.
2. **1초 만에 '내 얘기'로 만드는 공감 포인트**를 건다 — "제피가루?"(초보 배당 투자자의 낯섦), "현타 겪는 직장인"(타겟 구체화).
3. **약속의 크기**를 훅에서 먼저 던진다 — 004는 "딱 정해드릴게요"(결론 약속), 003은 "경제적 자유"(보상 약속).
4. **훅에서 구체적 상품명/숫자는 일부러 뒤로 뺀다** — 1차 훅은 감정/욕망 언어, 티커(SCHD/JEPI)는 훅 다음에 등장.

**인트로(훅 이후~본문 진입 전)의 공통 역할**:
- **끝까지 볼 이유** 3~4개 항목으로 쪼개 예고: 004 "왜 내가 S&P 500인지 + MBTI 유형 + 국내 상장 ETF 보너스", 003 "왜 인기인지 + 차이점 + 누구에게 맞는지", 002 "월 300 받으려면 얼마를 / 정확하게 / 알기 쉽게".
- **공포/기회비용 자극**: 004 "잘못 골랐다가 좋은 성과 낼 기회를 놓친다면 정말 아쉽겠죠" (FOMO).
- **진입장벽 낮추기 프레임**: 004 "MBTI처럼 딱 정해드릴게요", 001 "상식적으로 알아두고 가야겠다"(매수 추천 아닌 교양 프레임), 002 "정확하게 그리고 알기 쉽게".

**왜 효과적인가 (심리)**:
- 배당 ETF 시청자는 **이미 여러 영상을 본 상태**(검색 누적). 새 영상을 클릭할 때 "또 같은 얘기 아니야?"라는 경계심이 기본값 → 004가 압도적 조회수(67만)를 찍은 건 바로 그 피로를 훅 첫 문장에 담았기 때문.
- 배당 ETF는 **'장기 수익'이라는 추상적 이익**을 팔아야 해서 즉각적 보상 감각이 약함 → 003처럼 "제2의 월급", "경제적 자유"로 이익을 신체화해야 호흡이 잡힌다.

### 설계 원칙

**우리 ep07 훅 설계 체크리스트** (배당 ETF 시청자 전용):

1. **첫 문장은 시청자의 누적 피로/함정 경험을 언어화한다.**
   - 금지: "오늘은 배당 ETF에 대해 알아볼게요", "안녕하세요 스컹크입니다"
   - 권장: "배당률 12%라길래 샀더니 원금이 녹아있다? 이거 제 얘긴가 싶은 사람…", "SCHD 좋다길래 샀는데 3년째 제자리인 이유…" (함정을 먼저 건드린다)
2. **구체 티커(SCHD/JEPI)는 훅 2번째 문장 이후에 배치.** 1차 훅은 감정/함정/오해 언어.
3. **훅 직후 CTA 삽입** (채널 규칙): "이런 분석 놓치기 싫으면 구독 한번 눌러두고 시작하자" / "배당 ETF 시리즈 계속 이어가니까 알림 설정 해둬".
4. **인트로에서 3대 약속을 쪼개 예고** — (1) 배당률 숫자의 함정, (2) 상품별 손익 구조의 차이, (3) 내 상황별 처방. 004와 003의 "3개 질문 제시" 구조를 그대로 차용.
5. **ep06에서 던진 힌트(절세계좌 과세이연)를 훅 안에서 회수** — "지난번에 절세계좌 얘기하면서 배당 ETF는 과세이연하라고 말하고 끊었지. 그게 왜인지 오늘 파보자."로 시리즈 연속성 확보.
6. **003의 "현타 겪는 직장인" 타겟 좁히기**를 차용 — "근로 소득 말고 배당 소득 만들고 싶은 30~40대"로 인트로에서 타겟 명시.

**공통 서사 흐름 (4개 영상에서 반복)**:

**후킹 → 개념 → 비유 → 데이터 대비 → 반전 → 처방 → 회수**

- **후킹**: 시장 맥락 또는 시청자 욕망 언어화
- **개념**: 상품의 정체 (운용사/구조/분배주기)
- **비유**: 진입 장벽 해제 (매우 중요, 댓글 1순위 반응 장치)
- **데이터 대비**: 숫자를 **대비 쌍**으로만 제시 (X vs Y)
- **반전**: 시청자의 직관을 뒤집는 순간 (1년 vs 3년, 닷컴 14년, 커버드콜 녹음)
- **처방**: 상황별 분기 ("당신이 ~라면 A, ~라면 B")
- **회수**: 서두 비유를 마지막에 재호출

**전달 기법 공통 패턴**:

*비유 (가장 강력한 장치 — 댓글 히트 포인트 1순위)*:
- 001: 콜옵션 = "아이폰 23을 100만원에 살 티켓을 10만원에 사는 권리" / 제피 = "피난처"
- 003: **배당 함정 = "닭이 알을 매일 낳아서 좋았는데 알을 낳느라 정작 닭고기는 못 먹는 상태"** (회수형 비유, 댓글 히트)
- 003: 결론 = "몸집이 커지는 닭 vs 몸집은 똑같거나 작아지는 닭"
- 004: **ETF 3종 = 백반 한상 / 엽기떡볶이 / 할머니 콩나물국밥** (3회 반복 등장, 이탈 댓글까지 만든 강력한 앵커)
- 002: ETF = "100개 종목 볼륨대로 그대로 사는 것" / 배당 = "3개월에 한 번씩 월급" / 필수소비재 = "세제·식료품은 안 사진 않잖아요"

→ **공통**: (1) **생활 언어**로 치환 (음식/티켓/전화기/닭), (2) **1개 중심 비유를 영상 내 2~3회 반복 호출**(004의 음식 3종, 003의 닭/알), (3) 추상 경제 개념이 나올 때마다 배치.

*숫자 제시 방식*:
- **대비 쌍**이 기본: 003 "분배금 12.07% vs 3.61%", 002 "SP500 +33.7% vs SCHD +6%", 004 "보수 0.03% vs 0.2% vs 0.06%"
- **체감 환산**: 003 "천만원 투자하면 연 30만원 vs 120만원", 002 "9억 1,139만원 필요 / 월 150만 × 20년"
- **시간축 반전**: 003 "1년 결과 → 3년 결과 뒤집힘", 004 "닷컴버블 -81%, 회복 14년"
- **숫자 나열 금지**: 채널 프로필과 일치 — 숫자는 반드시 맥락과 대비 쌍으로.

*사례 활용*:
- 실제 인물/종목/사건으로 체감도 확보: 001 "JP모건", "KB/하나 배당 8%", "리츠 10%", "PF 부실" / 004 "엑슨모빌 탈락, 엔비디아/메타 부상" / 002 "찰스 슈왑", "본인 칼럼".
- **권위 소환**이 모든 영상에 존재: 001(JP모건), 002(찰스 슈왑 + 본인 저서), 003(찰스 슈왑), 004(**워런 버핏** — 썸네일 얼굴까지).

### 설계 원칙

**ep07 본론 뼈대 (반드시 이 순서)**:

1. **도입** (1~2분): 월배당 10%의 달콤함을 먼저 보여주고, **ep06에서 던진 과세이연 힌트**를 회수하며 "왜 배당 ETF인가" 시장 맥락 쌓기.
2. **개념/함정 도입** (2~3분): SCHD/JEPI의 정체 소개. 단, "좋은 상품 소개"가 아니라 **"배당률 숫자가 말하지 않는 것"** 프레임으로 열기.
3. **원리 해부 + 중심 비유 설치** (4~5분): 커버드콜 손익 구조를 001의 A/B 시뮬 방식 또는 003의 3시나리오 매트릭스로 재현. **이 단계에서 우리 영상의 '닭/알' 급 중심 비유 1개 확정**.
4. **데이터 대비** (3~4분): 대비 쌍으로만 제시. SCHD 분배금 3.6% vs JEPI 12% / 3년 총수익 SCHD vs JEPI vs SP500 / 천만원 기준 연 배당 금액.
5. **반전 전환 (핵심 리텐션 지점)**: "그런데 이 그래프를 3년으로 늘려보면…" 같은 시간축 반전 또는 "배당은 많이 받았는데 원금이 녹았다" 같은 NAV 반전 중 1개 필수.
6. **국내판 배당 ETF 카탈로그** (2~3분): TIGER 미국배당+7%/+3% 프리미엄, KODEX SP500 배당귀족 CC, KBSTAR 200 고배당 CC, TIGER 나스닥100 CC + 국내 SCHD 추종 ETF. 001이 짧게 훑은 지점 → **여기를 표로 깊게 파서 차별화**.
7. **처방 (상황별 분기)**: "월 현금흐름 급해? → JEPI, 은퇴까지 20년 남음? → SCHD, 둘 다 애매해? → 짬뽕 포트폴리오". 004의 MBTI 분기 + 003의 2택 프레임 혼합.
8. **회수 + 다음 영상 예고**: 서두 비유 재호출 + "ep08에서는 ~" (채널 규칙: 클로징에 재방문 유도).

**전달 기법 규칙**:
- **중심 비유 1개를 영상 시작/중간/끝에 3회 호출** (004의 음식 3종, 003의 닭/알 수준). 우리 후보: **"닭(성장) vs 알(배당) 트레이드오프"** — 이미 003이 검증한 프레임을 확장해, SCHD=몸집 커지는 닭, JEPI=알 많이 낳지만 몸집 줄어드는 닭, SP500=알 안 낳고 몸집만 키우는 닭으로 3종 분기.
- **숫자는 반드시 대비 쌍 또는 체감 환산**. "천만원 투자하면 연 30만원 vs 120만원 vs 0원" 같은 3축 비교.
- **추상 개념(커버드콜/NAV/총수익) 나올 때마다 생활 언어 1개 배치**.

### 설계 원칙

**썸네일 텍스트 설계 원칙 (ep07용)**:

4. **물음표·숫자·티커를 반드시 포함**:

7. **형식 템플릿 후보**:

**제목 설계 원칙 (ep07용)**:

**썸네일-제목 역할 분담 공식 (ep07)**:
- **썸네일** = 감정 자극 ("녹는다?", "함정?", "속지마") + 스컹크 경계 표정 + 티커/숫자 시각화

## ep07 차별화 지점 (레퍼런스가 빠뜨린 것)

레퍼런스 4개가 공통적으로 **덜 다뤘거나 실수한 지점** — ep07이 파고들 차별화 포인트:

1. **"국내판 배당 ETF 실부담비용 + 세금 심층 비교"** — 001이 짧게 훑고, 004가 일부만 다룸. TIGER 미국배당+7%/+3% 프리미엄, KODEX SP500 배당귀족 CC, KBSTAR 200 고배당 CC, ACE 미국배당다우존스, TIGER 미국배당다우존스 등 **국내 SCHD/JEPI 대응 상품을 표로 완전 비교**.
2. **"배당 vs 대체 선택지(부동산 월세, 채권, 성장 ETF)" 총수익 비교** — 002 댓글에서 "9억이면 재개발 아파트"가 터진 이유. 우리는 정면 비교.
3. **"커버드콜 NAV 녹음의 실제 증거"** — 001이 "녹는다"는 말만 하고 수치 증거는 부족. JEPI/QYLD의 **연도별 NAV 변화 + 배당 재투자 시 총수익** 실측.
4. **"ISA/연금 계좌에서 배당 ETF의 과세이연 효과"** — ep06과 연결. 일반 계좌 vs ISA/연금 계좌에서의 20년 복리 차이 시뮬.
5. **"사실 정확성"** — 004의 자동 리밸런싱 오류처럼 치명적 실수 금지. 특히 커버드콜 계산 방식, 분배율/배당률 구분, 기준가 NAV 설명에서 정확성 체크 필수.
6. **"배당 ETF를 '피해야 하는 사람'" 명시** — 모든 레퍼런스가 "누구에게 맞는지"만 다루고 "누구는 절대 사면 안 되는지"는 약함. ep07 주제("진실") 특성상 **부적합 시청자 명시**가 차별화.

**권장 차별화 포지션**: *"이 영상은 배당 ETF를 '팔려는' 영상이 아니라, 이미 샀거나 살까 고민 중인 사람이 **진짜 수익을 계산해볼 수 있는** 영상이다."*
