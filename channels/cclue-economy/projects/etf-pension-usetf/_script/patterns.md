## 4. 바이럴 제목 공식 (복제 가능)

```
[숫자/금액] + [구체 대상(나이·계좌)] + [반전/금기/방법]
```
글로벌 검증 공식:
- 구체 금액 + 따라하기 ("My Exact ETF Portfolio That Made...")
- 숫자 + 결과 ("Best 3 ETF Portfolio to make you...")
- 거짓말/폭로 + 종목명 ("The Covered Call ETF Lie Nobody Is Talking About — JEPI, XYLD")
- 시니어 나이 + 숫자 ("5 Reasons to File at 62")

⚠️ **안전 변환 필수** (13게이트): "망한다/사라/절대 사지마/평생 월 250만원 보장" → "갈리는 이유/확인할 점/노후 월배당 만드는 법" (수익 보장 단정·종목 reject·권위 인용 출처 회피)

## 5. 시니어 채널 성공 공식 (데이터 검증)

극소 채널 폭발 공통 요소:
- **구체 액수** (76조, 600억, 5억)
- **반전/금기 단어** (충격, 결단, 폭탄 — 단, 우리는 안전 표현으로)
- **권위 인용** (보고서·공식 발표 — 단, 출처 인라인 필수)
- **국가·세대 단위 사건** (은퇴자·연금 = 모두의 일)

## 6. 리텐션 장치 (본 영상 적용)

- 오픈루프: "그런데 이 ETF, 한국인이 놓치는 함정이 하나 있어" (파트 전환 직전)
- 2~3분 간격 재참여: 충격 수치 → 비유 → 시청자 질문(Q1 선언 택일) → 반전
- 스테이크 상승: 예적금 손해(개인) → 환율·세금 함정(구조) → 노후 30년 누적(인생)
- 감정 곡선: 불안 → 이해 → 안도 → 통제감

## 7. 본 영상 핵심약속 (제목→본문 정합)

제목이 약속하는 것 = "퇴직연금/은퇴자금을 미국 ETF·월배당으로 굴려 노후 현금흐름 만드는 구체적 방법". 본문 20% 이내(약 2분)에 핵심 해법(ETF 3종 + 계좌 활용)의 윤곽 제시 → 게이트 2(메타데이터 약속 이행) 충족.
