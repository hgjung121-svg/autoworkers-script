# 영상 컨셉

## 앵글
ep07 본편을 보면 자연스럽게 따라오는 의문 — "받은 배당은 어디 갔지" — 을 시청자가 묻기 전에 채널주가 먼저 깔아주는 보충편. "배당 재투자해도 결론은 같다"를 3트랙 정밀 시뮬로 증명하는 정밀 시뮬편. ep07 NAV 위주 분석을 받은 배당까지 다 합친 총수익 기준으로 한 단계 더 정밀하게 풀어주는 시리즈 자연 흐름. (patterns.md §A "한 단계 더 들어가는 보충편" + §"차별화 지점 7~10" 기반)

## 핵심 약속
이 영상을 끝까지 보면, JEPI 배당을 어디다 뒀느냐(생활비/같은 ETF/SP500)에 따라 6년 뒤 내 통장이 얼마나 갈리는지 1,000만원 기준으로 직접 보게 되고, "배당 재투자하면 결과 다른 거 아니냐" 의심이 풀리면서 본인이 박대리 A·B·C 중 누구인지 한 번에 결정 내리게 된다.

## 타겟 시청자
ep07 보고 "JEPI 비교가 좀 이상한데, 배당 재투자하면 다르지 않나?" 하고 한 번쯤 의문 들었던 사람 + 월 8만원짜리 JEPI 한 종목 들고 있으면서 "이거 진짜 잘 산 건가" 의심 끼인 직장인 + ep07-part2 처방 보고 "근데 내 비중 어떻게 잡아야 되지" 멈춰선 30~40대.

## 서사 유형
**미스터리 추적 + Before→After 대비 하이브리드** — "배당 재투자하면 결과가 정말 달라지나?"라는 자연 발문(미스터리)을 단서(JEPI 6년 DRIP)→가설(같은 ETF 재투자)→진실(SP500 재투자가 격차를 더 벌린다, 반전)로 추적. 동시에 박대리 A/B/C 6년 후 통장 잔고를 Before(2018년 1,000만원 동일 출발)→After(6년 뒤 격차)로 시각화. patterns.md §B-2 "박대리 캐릭터 분리"와 §A-2 "결론 보호 4회 호출"이 이 유형에서 자연스럽게 작동.

## 감정 전략
- 목표 감정: **신뢰 강화** (1순위, patterns.md §0 갱신) + **이해의 쾌감** (2순위, base 계승)
- 감정 아크: 자연 발문·정밀함(초반 5분) → 안도·궁금증("결론은 같다는데 어떻게?", 5~13분) → 충격·납득(SP500 재투자가 더 벌린다는 반전, 13~19분) → 통제감(샤프 비율로 "QQQ 10년 쉬운가" 풀이 + 박대리 트랙 자기 확정, 19~25분)
- 리텐션 접근: 미스터리 추적 유형은 정답까지 못 떠나는 특성을 활용. 닭/알 비유 4회 호출 + "셋 중 누가 가장 많을 거 같아?" 시청자 능동 참여 + 13:00 반전 전환("진짜 차이는 6년이 아니라 15년으로 늘려보면")으로 중반 이탈 봉쇄 (patterns.md §D-2 리텐션 8개 장치)

## 제목 후보

1. `"배당 재투자하면 다른 거 아니냐?" JEPI 6년 다시 계산했더니, 격차가 오히려 더 벌어졌다 (ep07 보충편)`
   - 패턴: 따옴표 문구(자연 발문 그대로 받기) + 변화/몰락 패턴 + 보충편 표시
   - 차용 레벨 L2: 003 패턴("그래프 더 늘려보면 얘기가 완전 달라집니다")의 반전 전환 톤 + ep07-part2 본편 후속 정체성
   - 효과: 시청자 자연 발문 어휘 그대로 받음 → "맞아 나도 그게 궁금했어" 즉시 진입. 본제목에 "더 벌어졌다" 반전 약속 명시.
   - 글자수: 49자 (따옴표 17자 + 본제목 32자, 50~60자 가이드 부합)

2. `JEPI 배당 6년치 어디다 뒀냐가 통장 7,621달러를 갈랐다 — 박대리 A·B·C 정밀 시뮬`
   - 패턴: 수치형 결과/증거 + 캐릭터 시리즈 표시
   - 차용 레벨 L4: base의 박대리 캐릭터 톤 + Total Real Returns 검증 수치 ($26,586 - $18,965 = $7,621)
   - 효과: 보충편 정체성은 약하지만 "내 통장 갈린다"는 개인 연결성 강함. 타겟 좁혀짐(이미 시리즈 본 시청자).
   - 글자수: 41자 (자유 형식 30~45자 부합)

3. `"제피 10년 뒤에도 돈이 거의 그대로?" 받은 배당 어디로 갔는지 풀어봤습니다 (ep07 보충편)`
   - 패턴: 따옴표 문구(자연 발문 어휘) + 비밀/진실 패턴 + 보충편 표시
   - 차용 레벨 L1: ep07 본편 보면 자연스럽게 따라오는 의문을 따옴표에 박음 ("10년뒤에도 돈이 거의 그대로라고?")
   - 효과: 가장 직접적인 정밀 풀이 톤. 거친 어휘는 "거의 그대로"만 살리고 톤은 부드럽게.
   - 글자수: 45자 (따옴표 18자 + 본제목 27자)

## 확정 제목 (사용자 최종 선택, 2026-05-10)
`"JEPI 10년 뒤에도 돈이 거의 그대로?" 형제 ETF QYLD 12년 데이터로 다시 계산했더니 (ep07 보충편)`

**선택 근거**:
- (a) ep07 본편 보면 자연스럽게 따라오는 의문 "10년 뒤에도 돈이 거의 그대로" 따옴표로 직접 받기 → "맞아 나도 그게 궁금했어" 즉시 진입
- (b) JEPI 자체는 2020년 상장이라 10년 데이터 자체가 존재하지 않음 → "형제 ETF QYLD 12년"으로 데이터 한계 정직 극복 + 장기 입증력 추가
- (c) QYLD 12년 DRIP +178.83% vs QQQ +780.10% (3.16배 격차) — 본편 6년 격차 76%p보다 더 강력한 증거
- (d) "(ep07 보충편)" 시리즈 자연 흐름 정체성 유지

**비교 분량 결정**:
- JEPI: 6년 DRIP (2020.5~2026, Total Real Returns 2025) — 단기 검증
- QYLD: 12.4년 DRIP (2013~2025, Total Real Returns 2025) — **장기 입증** (시간 갈수록 격차 더 벌어짐)
- QQQ/VOO/SCHD: 10년 누적 (Yahoo Finance Adjusted Close)
- 분배율 20% 시뮬: 한국경제 2025.01.20 (NAV 기준 가상 시뮬, "옵션 매도 효과 미반영" 명시)

## 썸네일 이미지 컨셉

### 컨셉 A: 똑똑이가 계산기 들고 정밀 표정 — "한 단계 더 정밀하게 풀어준다" 톤
- 구도 패턴: 인물 감정(중앙 집중) + 상단 47% 스컹크 + 하단 53% 검정
- 핵심 오브젝트: 똑똑이 스컹크(REACT-05 교수 모드 변형 — 한 손에 계산기, 다른 손 검지 치켜 "포인트 짚는" 제스처) + 옆에 떠 있는 종목 티커 "JEPI" 글로우
- 색채/대비: 어두운 네이비(#1a1a2e) → 검정 그라데이션 배경 + 골드 포인트(브랜드 컬러) + JEPI 티커 빨강 글로우
- 감정 톤: 차분한 권위 + 정밀한 풀이 (긴장 + 신뢰 양립)

### 컨셉 B: 박대리 A vs C 통장 잔고 대비 — "받은 배당 어디 갔냐"
- 구도 패턴: A vs B 좌우 분할 (좌 박대리 A 통장, 우 박대리 C 통장) + 똑똑이가 가운데서 손가락질
- 핵심 오브젝트: 똑똑이 스컹크(REACT-04 변형 — 양손으로 격차 표현 제스처) + 좌우 통장/달러 아이콘 + 중앙 "76%p" 숫자 글로우
- 색채/대비: 좌측(박대리 A) 어둡게 / 우측(박대리 C) 골드 라이트 / 배경 네이비 → 검정 그라데이션
- 감정 톤: 충격 대비 + 시각적 드라마 (Before→After 패턴)

## 선택 근거
3개 패키지 중 (1) 보충편 정체성 + (2) 반전 약속 + (3) 시리즈 자산화 3박자가 가장 강함. patterns.md §A-1·A-2(자연 발문+결론 보호), §A-4(보충편 표시), §B-2(반전 전환), §0 신뢰 강화 1순위 감정과 일관됨. Total Real Returns 1차 출처 검증 수치(JEPI DRIP +89.65% / SPY DRIP +165.86%, 76%p 격차)를 썸네일·제목에 그대로 인용해 게이트 4(YMYL) 출처 인라인 강제 룰 충족. ep07 본편 시청 후 자연 발문이 1차 자산이라는 점에서 ep07-part2 base patterns의 "조회수 1위 레퍼런스 차용" 원칙(004 67만 뷰 패턴)을 시리즈 내부 자산으로 대체.
