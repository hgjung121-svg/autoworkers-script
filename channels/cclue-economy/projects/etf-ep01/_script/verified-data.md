## 1. 핵심 수치 — 대본 직접 인용용

### 수수료 복리 시뮬레이션 (핵심 장면)
```
조건: 월 50만 원 × 30년, 세전 연 수익률 7%
원금 합계: 1억 8,000만 원

ETF (보수 0.05%)  → 약 5억 8,500만 원
펀드 (보수 1.5%)  → 약 4억 4,700만 원
────────────────────────────────
차이: 약 1억 3,800만 원

→ 수수료 1.45%p 차이가 30년간 약 1.4억 원을 갉아먹는다.
→ 같은 시장에 투자했는데, 수수료만으로 자산의 약 24%가 사라진다.
```

### 20년 기준 시뮬레이션
```
조건: 월 50만 원 × 20년, 세전 연 수익률 7%
원금 합계: 1억 2,000만 원

ETF (보수 0.05%)  → 약 2억 6,200만 원
펀드 (보수 1.5%)  → 약 2억 2,300만 원
────────────────────────────────
차이: 약 3,900만 원
```

### 비유용 한 줄
> "30년간 매달 50만 원씩 투자하면, 수수료 차이만으로 중형차 한 대 값(1.4억)이 증발합니다."

|------|------------------|

### 차이나 펀드 사례 (2007~2008)
- 2007년 한국에서 차이나 펀드 가입 열풍 — 은행 창구에서 적극 권유
- 상하이종합지수: 2007.10 → 6,124pt (사상 최고) → 2008.11 → 1,664pt (**-72.8%**)
- 한국 투자자 수십만 명, 수조 원 규모 손실
- 대부분 TER 2% 이상의 고비용 펀드
- 손실이 나도 수수료는 매년 자동 차감 → 투자자 분노

### 대본용 핵심 메시지
> "2007년, 은행에서 '중국이 뜬다'며 권유한 펀드에 수십만 명이 가입했습니다. 수수료 2%. 1년 뒤, 투자금의 70%가 사라졌습니다. 그런데 운용사는 그 1년간에도 수수료를 꼬박꼬박 챙겼습니다."

## 추가 리서치 데이터

### ETF의 구조적 장점 (대본 보조 자료)
| 장점 | 설명 |
|------|------|
| **낮은 비용** | 인덱스 추종으로 리서치/매니저 비용 불필요 → TER 0.03~0.10% |
| **투명성** | 보유 종목 매일 공개 (뮤추얼펀드는 분기별) |
| **유동성** | 주식처럼 실시간 매매 가능 (뮤추얼펀드는 1일 1회 기준가) |
| **세금 효율** | 미국의 경우 in-kind creation/redemption으로 자본이득세 최소화 |
| **분산투자** | 1주로 수백~수천 종목에 분산 |

### 역사적 맥락: 왜 액티브가 지는가?
1. **효율적 시장 가설**: 공개 정보는 이미 가격에 반영 → 초과 수익 어려움
2. **수수료 부담**: 매년 1.5% 차감은 복리로 누적되어 치명적
3. **생존편향**: 성과 나쁜 펀드는 합병/폐쇄 → 살아남은 펀드만 보면 착각
4. **회귀 효과**: 단기 호성적 펀드매니저도 장기로 보면 평균 회귀
5. **규모의 저주**: 펀드 규모가 커지면 시장 충격 없이 매매하기 어려움
