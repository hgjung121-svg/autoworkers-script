**공통 패턴 3가지:**

1. **손실 프레이밍이 이익 프레이밍을 압도한다** — 8개 중 8개가 "안 하면 망한다/반토막/손해/개박살"로 시작. "이렇게 하면 좋다"는 단 하나도 없다.
2. **구체적 수치를 첫 5초 안에 투하한다** — 조회수 상위 3개(006, 009, 008) 모두 훅에서 "조회수 10", "10회 100회", "0명" 등 구체 숫자를 즉시 언급. 추상적 "어렵다"가 아니라 숫자로 고통을 고정한다.
3. **시청자의 고통을 '대신 말해주는' 공감형 독백** — 상위 영상일수록 질문("안 나오시죠?")보다 독백("아, 이제 알아서 터지겠지")을 사용. 시청자의 내면 대화를 재현하는 방식이 더 강한 동일시를 만든다.

**인트로 공통 역할:**

8개 영상 전부 인트로에서 3가지를 수행한다:
- **신뢰 근거 즉시 제시**: "천 권 분석"(004), "24만/21만/7만 채널 적용"(009), "38만 유튜버 비법"(008)
- **손실 회피 강화**: "이걸 안 하면 손해"(002), "알고리즘이 버린다"(004), "100% 손해"(005)
- **완결성 약속**: "처음부터 끝까지"(002), "10가지 모두"(009), "이 영상으로 끝"(006)

**공통 서사 흐름:**
> **문제 진단(왜 안 되는가) → 원리 설명(메커니즘) → 해결책 단계별 전달 → 결과 재확인**

나열형이 아니라, 반드시 "왜"가 먼저 오고 "어떻게"가 뒤따른다. 8개 모두 이 순서를 지킨다.

**전달 기법:**

*비유*:
- "간판도 메뉴판도 없는 식당" (004) — 추상 개념을 1초 만에 이해시키는 일상 비유
- "도서관 서적 카테고리 분류" (005) — 기술 개념을 익숙한 물리 공간으로 치환
- "유튜브는 자선 단체가 아니잖아요" (008) — 플랫폼 본질을 한 문장으로 재정의
- 공통점: **추상적 개념이 등장할 때마다 비유를 즉시 배치**한다. 설명 후 비유가 아니라 비유가 설명을 대체한다.

*숫자*:
- 8개 영상 전부 숫자를 **대비 쌍**으로 제시: "조회수 30 → 102만"(002), "12회 → 150만"(003), "6.6천 → 22.7만"(006), "조회수 7 → 255.3만"(008), "ON=0명 vs OFF=7만"(005)
- 단독 숫자("매출 43% 하락")보다 Before/After 대비("X에서 Y로")가 압도적으로 많다.

*사례*:
- 상위 영상일수록 **본인 실제 데이터를 공개**한다: 006("클릭률 1.5~2배"), 007("139만/174만 조회수 + 수익 금액"), 008("38만 채널")
- 남의 사례가 아니라 자기 경험이 신뢰를 만든다.

**리텐션 장치 배치 패턴:**

| 위치 | 장치 | 출현 빈도 |
|------|------|----------|
| 초반(훅 직후) | 오픈루프 — "더 무서운 이유가 있다", "아직 준비 과정을 놓치고 있다" | 8/8 |
| 중반(파트 전환) | 패턴 인터럽트 — "솔직히 말씀드릴게요"(004), "자, 이제 기본기는 끝났습니다"(006) | 7/8 |
| 중반~후반 | 전환 멘트(다음 파트 예고) — "여기서부터가 진짜입니다" | 6/8 |
| 파트 말미 | 공포→안도 리셋 — 위협 후 해법 제시, 다음 파트에서 새로운 위협 재시작 | 5/8 (특히 007이 전형적) |

### 설계 원칙 (깊이 읽는 경제 적용)

| # | 규칙 |
|---|------|
| B1 | **본론의 뼈대는 "왜 위험한가(진단) → 메커니즘(원리) → 그래서 어떻게(대응)"의 3단 구조를 따른다.** 사실 나열 금지. 경제 현상도 반드시 인과로 연결한다. |
| B2 | **추상적 경제 개념이 나올 때마다 1초 비유를 즉시 배치한다.** "양적완화 = 돈을 찍어서 시장에 풀어넣는 것" 수준이 아니라, "불난 집에 물 대신 기름을 붓는 격이야" 수준의 감정이 실린 비유. |
| B3 | **숫자는 반드시 대비 쌍(Before/After, X vs Y)으로 제시한다.** "금리가 3.5%다"(약함) → "금리 1.75%일 때 월 이자 42만 원이었는데, 3.5%가 되니까 84만 원이야"(강함). |
| B4 | **파트 전환 직전에 오픈루프를 설치한다.** "여기까지가 1차 충격이야. 근데 진짜 문제는 다음이거든." |
| B5 | **파트마다 공포→안도 사이클을 반복한다.** 한 파트에서 위기를 제시하고 해법으로 안도시킨 뒤, 다음 파트에서 새로운 위기를 다시 시작. 영상 전체가 롤러코스터 구조를 형성하게 한다. |
| B6 | **신뢰 데이터는 남의 것보다 직접 확인한 것을 쓴다.** "한국은행 발표에 의하면"보다 "한국은행 보고서 36쪽을 직접 열어봤는데, 여기 이 그래프를 보면..." |

| 감정 | 대본 속 장치 (원문) |
|------|-------------------|
| 공포/손실 | "이거 체크 잘못하면은 나중에 수익이 안 들어옵니다" (002) / "알고리즘이 싫어하는 채널은 끝까지 버려진다" (004) / "방금 당신은 당신의 채널을 스스로 죽였습니다" (007) |

**감정 흐름 공통 패턴:**

8개 영상의 감정 곡선을 겹쳐보면 공통 구조가 드러난다:

```
초반 30% : 공포/박탈감/억울함 (고통 축적)
중반 40% : 공포→안도 사이클 반복 (파트마다 리셋)
후반 30% : 희망/실행 의지/안도 (보상)
```

핵심: **중반에 감정이 일직선으로 올라가지 않는다.** 파트마다 새로운 위협으로 공포를 리셋하고 해법으로 안도시키는 사이클이 반복된다. 007이 가장 전형적 — "공포(훅) → 억울함(지인 공유) → 안도(딥링크) → 공포(프로모션) → 안도(체크리스트) → 공포(아동용) → 안도(설정법) → 공감(클로징)".

### 설계 원칙 (깊이 읽는 경제 적용)

| # | 규칙 |
|---|------|
| E1 | **초반 30%는 고통을 충분히 축적한다.** 바로 해법을 주지 않는다. "이게 왜 위험한지"를 수치와 사례로 쌓아서 시청자가 "이거 나한테도 해당되는 거 아니야?"라고 불안해질 때까지 끌고 간다. |
