# 레퍼런스 분석 — 002

**영상**: 조회수 안 터지는 진짜 이유! 업로드 후 꼭 해야 할 설정 18가지 (99%가 안함) | 2026년 최신 기준
**채널**: 막내혀누 | 40대이상 유튜브 기초과정 (구독자 4,860명 / 조회수 26,387)

---

## 1. 주제 요약

유튜브 왕초보(40대 이상)가 업로드 전후에 반드시 해야 할 채널 세팅 18가지를 몰라서 조회수를 놓치고 있다.

## 2. 썸네일 & 제목 분석

### 썸네일 텍스트 추출

| # | 텍스트 (원문 그대로) | 글자수 | 크기/색상/위치 | 확신도 |
|---|-------------------|-------|-------------|-------|
| 1 | `조회수` | 3자 | 최대 폰트, 노란색, 좌측 상단 | 확실 |
| 2 | `터지는` | 3자 | 최대 폰트, 빨간색, 좌측 중상단 | 확실 |
| 3 | `업로드 세팅` | 6자 | 대형 폰트, 흰색, 좌측 중단 | 확실 |
| 4 | `이대로만 하세요` | 8자 | 대형 폰트, 노란색, 좌측 하단 | 확실 |
| 5 | `조회수 30` | 5자 | 소형, 흰색, 중앙 상단 박스 안 | 확실 |
| 6 | `조회수 102만` | 7자 | 소형, 빨간 점선 박스 안, 우측 상단 | 확실 |

### 썸네일 비주얼 분석

| 항목 | 분석 |
|------|------|
| 구도 패턴 | Before→After (조회수 30 → 조회수 102만) 수치 대비 |
| 핵심 오브젝트 | 조회수 30 박스 / 조회수 102만 빨간 점선 박스 (화살표로 연결) |
| 색채 | 배경: 진한 초록색, 포인트: 빨간색·노란색, 브랜드컬러: 없음 |
| 인물 | 있음, 중년 남성, 웃는 표정, 시선 정면 |
| 텍스트 공간 | 좌측 70% 차지, 계층형 배치 |
| 시각 장치 | 화살표(30→102만), 빨간 점선 박스(강조), 두 개 수치 박스 대비 |
| 감정 | 기대감(나도 저렇게 될 수 있다) |

- 텍스트 레이아웃 요약: 총 6개 텍스트 요소, 좌측 계층형(메인 카피) + 우측 수치 대비(Before/After) 복합 구조

### 제목 분석

- 제목 전문: `조회수 안 터지는 진짜 이유! 업로드 후 꼭 해야 할 설정 18가지 (99%가 안함) | 2026년 최신 기준`
- 핵심 장치:
  - **역발상 진단**: "터지는 방법"이 아니라 "안 터지는 진짜 이유"로 시작 → 문제 원인 먼저 제시
  - **구체적 숫자**: "18가지" → 볼 가치가 명확히 정의됨
  - **사회적 증명 + 희소성**: "(99%가 안함)" → 나만 모르고 있다는 공포 유발
  - **시의성**: "2026년 최신 기준" → 구식 정보 불신 해소

### 썸네일-제목 조합이 만드는 기대감

"이 영상을 클릭하면 조회수 30짜리 채널을 102만으로 만드는 구체적인 설정 방법을 알 수 있을 것 같다"

## 3. 훅 & 인트로 분석

### 훅 (첫 3~5초 상당 분량)

> "자, 근데 조회수 없음 또는 5, 10 미만이신 영상들이 대부분이다 하시면 안 하는 걸 저는 권해드립니다."

- **어떤 심리를 건드렸는가**: 조건부 거절("이건 당신한테 안 맞아")로 역설적으로 호기심을 자극한다. "그럼 나는 해야 하나 말아야 하나?"를 먼저 판단하게 만들어 시청자가 자기 채널을 기준으로 즉시 관여하게 만든다. 패턴 인터럽트 + 역설적 선언 유형.
- **효과적인 이유**: 일반적인 "이거 꼭 하세요"와 반대로 "하지 마세요"를 먼저 던져 신뢰를 형성하면서, 동시에 "그럼 나는 어느 쪽이지?" 자기분류 욕구를 자극한다.

### 인트로 (훅 이후 ~ 본문 진입 전)

> "오늘 말씀드릴 내용은 정말 중요한 내용입니다. 처음 유튜브 시작하는 분들이 이걸 하지 않아서 손해 보는 경우가 너무 많습니다. ... 이번에 유튜브 내에서 새로 바뀐 내용들이 굉장히 많기 때문에 오늘은 2026년 최신 버전으로 제가 처음부터 끝까지 다 알려 드릴 테니까요. 꼭 영상 끝까지 시청하셔서 2026년도 조회수 대박 나시길 바라겠습니다."

- **인트로의 역할**: 세 가지 끝까지 봐야 할 이유를 동시에 제시한다. ①"안 하면 손해"(공포), ②"2026년 최신 버전"(시의성), ③"처음부터 끝까지 다 알려준다"(완결성 약속).
- **평가**: 혜택(대박)과 공포(손해)를 동시에 쓰는 강한 인트로. "오늘은 ~에 대해 알아볼게요" 유형이 아니라, 결과(조회수 대박)와 손실 회피(손해 보는 경우가 너무 많다)를 선명하게 건드린다.

## 4. 본문 구조 분석

### 전체 구조 요약

1. **쿠키 싹기(채널 존재 알리기)**: 유튜브가 채널을 분석하는 데 24시간~1주일 걸리며, 비슷한 채널 10~20개 구독으로 "사람임을 증명"
2. **크롬 확장 프로그램 2종 설치**: 유튜브 서머리(대본 추출), VDQ(조회수 점수 분석) → VPH 500 이상 기준 제시
3. **제목·썸네일 키워드 전략**: 유튜브 스튜디오 트렌드 탭 활용, 시크릿 모드 검색(Ctrl+Shift+N)
4. **채널 기본 설정**: 설명 문구 SEO, 해시태그, 이메일, 워터마크, 거주 국가, 아동용 설정("이거 체크 잘못하면은 나중에 수익이 안 들어옵니다")
5. **기능 사용 자격 요건**: 표준 → 중간 기능(핸드폰 인증, 15분 이상 영상·커스텀 썸네일 활성화) → 고급 기능(AB 테스트)
6. **업로드 기본 설정**: 파일명, 재생목록, AI 콘텐츠 표기, 태그 오타 키워드 전략
7. **엔딩 화면 설정**: 영상 1개 + 구독 버튼 조합
8. **예약 업로드**: 최소 1시간 후 예약 → 유튜브 분석 시간 확보 + 업로더 본인 최종 검토

**정보 배치 패턴**: 문제 진단 → 도구 장착 → 채널 세팅 → 영상 업로드 세팅 → 마무리 점검. "집을 지을 때 땅을 파고 기초를 세우는 과정"이라는 비유로 전체 구조를 하나의 인과로 묶음.

### 전달 기법

- **스토리텔링**: 없음. 나열형 실습 튜토리얼 구조.
- **비유**: "작은 콩 하나 나눠 먹는 그런 개념도 아니고" (조회수 없을 때 AB 테스트가 무의미하다는 것을 시각화), "집을 지을 때 땅을 파고 기초를 세우는 가장 중요한 과정"
- **데이터/숫자**: VPH 500 이상 기준, 구독 최소 10~20개, 시청 최소 1시간, 100회 미만이면 AB 테스트 불필요, 태그 약 20개
- **사례**: 크롬 확장 프로그램 실시간 화면 시연("138K VPH 100X" 실제 수치 제시)

### 리텐션 장치

- **오픈루프 (초반)**: "유튜브 내적인 설정만 중요한게 아닙니다. 유튜브를 시작하기 전에 준비 과정도 굉장히 중요한데 많은 분들이 또 그걸 놓치고 계시더라고요." → 아직 밝히지 않은 준비 과정에 대한 궁금증 유발
- **경고 장치 (중반)**: "이거 체크 잘못하면은 나중에 수익이 안 들어옵니다. 그래서 이게 제일 중요합니다." → 단순 정보 나열에서 위기감으로 전환, 집중도 회복
- **전환 멘트 (후반)**: "자 이제 대망의 마지막 순서입니다." → 완결 신호로 이탈 방지

## 5. 감정 트리거

**주요 감정**: 공포(손해 보는 공포) → 기대감(나도 할 수 있다) → 안도(친절하게 다 알려준다)

| 감정 | 유발 장치 | 위치 |
|------|----------|------|
| 공포/손실 회피 | "처음 유튜브 시작하는 분들이 이걸 하지 않아서 손해 보는 경우가 너무 많습니다" / "이거 체크 잘못하면은 나중에 수익이 안 들어옵니다" | 인트로·중반 |
| 기대감 | 썸네일 "조회수 30 → 102만" 수치 Before/After | 클릭 전 |
| 안도/신뢰 | "제가 최대한 알기 쉽게 설명을 드린다고 했으나 혹시라도 혼자서 하기가 힘들거나... 일대일로 직접 구독자님을 찾아가서 무료 강의를 해 드리고 있으니까요" | 후반 |

**감정의 흐름**: 공포(내가 이걸 안 하고 있었나?) → 기대(배우면 되겠다) → 안도(이 분 믿어도 되겠다) → 행동 유도(댓글로 지역 남기기)

## 6. 댓글 반응 분析

### 시청자가 가장 강하게 반응한 포인트

- **좋아요 29개 댓글**: "저는 80살 유투브 왕초보인데 배우면서 하는데 정말 잘했다고 생각해요 아주 재미있어요" → 80세 시청자가 유용하다고 반응. 이 채널의 타겟("40대 이상")이 실제로 훨씬 넓은 연령대를 포괄함을 보여준다.
- **좋아요 6개 댓글**: "이제껏 이렇게 끼지 시원하게 잘 알려주신 분은 없었다" → "시원하게"라는 표현 = 그동안 다른 곳에서 이 정보를 명확히 얻지 못했다는 미충족 수요.
- 반복되는 감사 댓글("친절하고 자세하게", "알기 쉽게 차근차근")들은 모두 **전달 방식의 명확성**에 반응. 정보 자체보다 "쉽게 설명해준다"는 점에 공감.

### 시청자 포지션

유튜브를 막 시작했거나 시작하려는 40대 이상 초보자. 정보는 있지만 어디서 어떻게 따라해야 하는지 몰라 막혀 있는 상태. 신뢰할 수 있는 "길 안내자"를 찾고 있음.

### 우리 영상에서 반드시 다뤄야 할 포인트

- "이렇게 시원하게 알려주신 분은 없었다"는 댓글은, 경제 채널에서도 "이 현상을 이렇게 명확하게 해석해준 곳이 없었다"는 반응을 끌어낼 수 있다면 강력한 리텐션이 된다는 것을 시사한다.
- 정보의 난이도보다 "설명의 친절함"이 구독·좋아요로 이어진다. 경제 개념을 쉽게 푸는 것이 단순 정확성보다 더 높은 감사 반응을 만든다.

## 7. 핵심 약속 역추적

### 제목+썸네일이 약속한 것

"이 영상을 보면 조회수 30짜리 채널이 102만이 되는 업로드 세팅 방법을 처음부터 끝까지 알게 될 것 같다"

### 인트로에서 확인해준 것

> "오늘은 2026년 최신 버전으로 제가 처음부터 끝까지 다 알려 드릴 테니까요. 꼭 영상 끝까지 시청하셔서 2026년도 조회수 대박 나시길 바라겠습니다."

제목의 약속(설정 → 조회수 대박)을 그대로 재확인. "처음부터 끝까지 다"라는 표현으로 완전성 약속을 추가.

### 본문이 실제로 전달한 것

유튜브 채널 개설부터 업로드 예약까지의 전 과정에서 놓치기 쉬운 설정 항목을 화면 시연과 함께 단계별로 설명하는 실용 가이드.

### 약속 이행 평가

- **이행 여부**: 부분 이행. "조회수 102만"의 비결이라는 기대를 주지만 실제로는 채널 세팅 체크리스트이며, 102만 사례와의 직접적인 인과 연결 설명은 없다.
- **오버딜리버리 여부**: 있음. 일대일 무료 방문 강의 제공(댓글로 지역 남기기)은 영상 내 약속 외의 추가 가치로, 강한 신뢰 형성과 댓글 유도를 동시에 달성.
