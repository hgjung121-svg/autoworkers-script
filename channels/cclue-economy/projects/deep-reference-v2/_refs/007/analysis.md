## 1. 주제 요약

## 3. 훅 & 인트로 분석

### 훅 (첫 3~5초 상당)

> "제발 아무것도 하지 마세요. 어쩌면 여러분은 여러분 스스로 영상을 개박살내고 있는지도 모릅니다."

- **유형**: 패턴 인터럽트("뭐야?") + 공포형("나도 당하나?")
- "아무것도 하지 마세요"는 직관에 완전히 반하는 명령형 문장. 콘텐츠를 만드는 사람에게 "하지 마라"는 금지령은 즉각적인 멈춤 효과를 만든다.
- "개박살내고 있는지도 모릅니다"는 현재 진행형 공포. 이미 돌이킬 수 없는 피해가 진행 중일 수 있다는 긴박감을 심는다.

### 인트로 (훅 이후 ~ 본문 진입 전)

> "영상을 만들어 업로드했다면 이 아홉 가지는 절대 하지 마세요. 첫 번째 구걸하지 마세요."

- **강한 인트로**로 평가. 곧바로 번호를 붙여 리스트를 예고함으로써 "끝까지 봐야 전체를 얻는다"는 완결 욕구를 자극한다.
- "이 아홉 가지"라고 먼저 예고하고 실제로는 다섯 가지만 다루는 구조 — 의도적 불완전 공개로 다음 영상 유인 장치도 겸한다.

## 4. 본문 구조 분석

### 전체 구조 요약

1. **지인 공유 금지** — 알고리즘 데이터 오염 원리 설명, 엄마-임영웅 팬 비유
2. **딥링크 대안 제시** — 금지만 하지 않고 올바른 홍보법(재생목록 링크)을 즉시 제공
3. **프로모션 금지** — "돈 내고 채널을 죽이는 뻘짓" 프레임, 139만·174만 조회수 자신의 데이터 증거
4. **즉시 공개 금지** — 화질 SD 처리 과정, 알고리즘 학습 시간 원리. 6단계 업로드 프로세스 제시
5. **아동용/연령제한 설정 금지** — 댓글 삭제·알림 중단·수익 저하 등 구체적 불이익 열거
6. **구독자 구매 금지** — "진짜 47명이 진짜 유튜브"로 감정 마무리

**정보 배치 패턴**: 금지령 선언 → 원리 설명(왜 망하는지) → 대안 제시 → 다음 금지령으로 전환
각 파트가 동일한 리듬으로 반복되어 예측 가능한 학습 구조를 만든다.

### 전달 기법

- **스토리텔링**: "우리 엄마에게 영상을 보내면 어떻게 될까요?" — 가상의 상황을 시나리오로 구체화해 알고리즘 개념을 체감 가능하게 전달
- **비유**: "여러분이 힘들게 홍보해서 남은 일만 시킨 꼴", "이탈할 틈을 아예 차단", "감옥 아 채널에 갇치게 됩니다" — 추상적 알고리즘 개념을 일상 언어로 번역
- **데이터**: "탐색 기능으로 온 사람이 60%, 추천 동영상 34%, 직접 입력 3%", "640만 원", "1,290만 원" — 본인 채널 실제 데이터를 증거로 활용
- **사례**: 본인의 139만·174만 조회수 영상을 직접 언급, 수익 금액까지 공개하여 권위 확보

### 리텐션 장치

- **오픈루프 (초반)**: "이것보다 더 무서운 이유가 있습니다. 바로 데이터 오염." — '더 무서운 것'을 예고해 이탈 방지
- **오픈루프 (중반)**: "화질보다 더 무서운 이유가 있습니다. 바로 알고리즘의 학습 시간입니다." — 같은 패턴 반복으로 "다음에 뭐가 나올지" 기대를 지속 유지
- **전환 멘트 (중반)**: "자, 이제 화질도 챙겼고 알고리즘도 네 편으로 만들었습니다. 하지만 이런 기다림도 지금부터 알려 드릴 두 가지 설정이 켜져 있다면 아무 짝이도 쓸모가 없습니다." — 앞 내용을 요약하고 다음 위험을 예고하는 브릿지
- **패턴 인터럽트 (각 파트 전환)**: "절대 하지 마세요. 두 번째 프로모션은 절대 클릭하지 마세요." — 매 파트를 새로운 경고로 시작해 주의를 환기

## 5. 감정 트리거

**주요 감정**: 공포 → 안도 → 공포 (반복 사이클)

| 감정 | 유발 장치 |
|------|---------|
| 공포 | "방금 당신은 당신의 채널을 스스로 죽였습니다" — 이미 실수했을 수 있다는 회고적 공포 |
| 억울함 | "여러분이 힘들게 홍보해서 남은 일만 시킨 꼴입니다. 억울하지 않으신가요?" — 직접 감정 명명 |
| 안도 | 각 파트 말미의 대안 제시 (딥링크, 업로드 6단계 체크리스트, 스튜디오 설정법) |
| 공감 | "구독자 47명. 부끄러워하지 마세요. 그 명은 당신의 영상을 진짜로 좋아해서 누른 진짜들입니다." |

**감정의 흐름**: 공포(훅) → 억울함(지인 공유 파트) → 안도(딥링크 해법) → 공포(프로모션) → 안도(체크리스트) → 공포(아동용/연령제한) → 안도(설정법) → 공감/동기부여(클로징)

공포와 안도가 교대로 반복되는 구조. 시청자를 위협한 뒤 해법으로 구제하는 사이클이 리텐션의 핵심 엔진이다.

## 6. 댓글 반응 분석

### 시청자가 가장 강하게 반응한 포인트

- **댓글 3위 (좋아요 23)**: "이 내용 300만원짜리 강의에서 엄청난팁이라고 알려줍니다 고액강의듣지말고 발품팔아서 고품질교육영상 찾아서 공부하세요"
  - 반응 포인트: 콘텐츠 자체의 정보 밀도. 유료 강의 수준의 정보를 무료로 얻었다는 가성비 인식
- **댓글 2위 (좋아요 5)**: 업로드 전 세팅 체크리스트를 댓글에 그대로 정리해서 작성 — 실용적 정보에 대한 강한 공감 반응
- **댓글 6위**: "제가 완전히 잘못 알고 있던 부분을 잘 짚어주셨네요" — '오해 교정' 포인트에 반응

### 시청자 포지션

유튜브를 이제 막 시작했거나 시작을 고민 중인 초보 크리에이터. 조회수가 안 나오는 이유를 몰라서 답답한 상태. 고액 강의를 경계하면서도 검증된 정보를 원하는 사람들.

### 우리 영상에서 반드시 다뤄야 할 포인트

- **"유료 강의 정보를 무료로"** 프레임이 유효함 — 경제 분야에서도 "증권사 리포트/전문가 분석을 쉽게 해설" 포지셔닝과 연결 가능
- 시청자는 **구체적 체크리스트/행동 가이드**에 강하게 반응 — 단순 분석보다 "오늘부터 바꿀 수 있다"는 실행 가능한 정보 포함 시 댓글 참여율 상승
- **통념 뒤집기** 포인트가 댓글 반응을 이끌어냄 — 경제 영상에서도 상식으로 알려진 것이 실제로는 틀렸다는 역설 구조가 유효

## 7. 핵심 약속 역추적

### 제목+썸네일이 약속한 것

"이 영상을 보면 유튜버 대부분이 저지르는 알고리즘을 망치는 행동들과 그 진짜 이유를 알게 될 것 같다"

### 인트로에서 확인해준 것

> "영상을 만들어 업로드했다면 이 아홉 가지는 절대 하지 마세요."

제목의 약속과 일치하며 더 구체화. "94%의 실수" → "9가지 금지 리스트"로 형태가 명확해졌다. 단, 실제로는 5가지만 다루고 나머지는 미공개 처리 — 의도적 불완전 이행으로 다음 영상 유도.

### 본문이 실제로 전달한 것

지인 공유, 프로모션, 즉시 공개, 아동용 설정, 구독자 구매 — 5가지 실수의 원리와 대안을 실제 데이터와 함께 제공하여, 시청자가 내일 당장 적용 가능한 실행 지식을 전달했다.

### 약속 이행 평가

- **부분 이행** — "9가지"를 약속했으나 5가지만 공개
- **오버딜리버리** — 금지 이유(알고리즘 작동 원리)와 대안(딥링크, 6단계 체크리스트, 스튜디오 설정)까지 세트로 제공. 단순 금지 목록이 아니라 원리 이해 + 즉시 실행 가능한 가이드로 약속 이상의 가치 전달.
