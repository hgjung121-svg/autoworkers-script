## 3. 훅 & 인트로 분석

**훅**

> "만약 여러분이 구독자나 조회수가 오르지 않는다면 아마도 이것을 끄지 않아서 그럴 것입니다"

- 심리: 패턴 인터럽트 + 공포형. '이것'을 숨겨 호기심 자극. 문제 원인을 외부(모르던 설정)에 귀인시켜 자책 대신 해결 욕구를 불러일으킴.

**인트로**

> "정말 오랜 시간 영상을 만들고 썸네일을 제작해서 업로드했는데 구독자는 오르지 않고 영상 조회수는 내가 본 1회 뿐인 분들 정말 많이 계시죠"
> "백날 제가 알려드린 영상 제작법 똑같이 따라서 올려도 이거 안 하시면 말짱 도루묵이 끝까지 안 보시면 100% 손해입니다 딱 3분만이 영상에 투자해서"

- 공감 → 공포 → 혜택 3단 구조. "말짱 도루묵", "100% 손해"로 이탈 비용 극대화, "딱 3분"으로 실행 허들 제거.

## 4. 본문 구조 분석

1. 세팅 1 — 채널 키워드 입력 (채널 정체성 등록)
2. 세팅 2 — 아동용 설정 OFF (오해로 잘못 설정하는 사례 경고)
3. 세팅 3 — 업로드 기본값 + 태그 (고정값 방치 금지, 재생목록 설정)

구조 패턴: 문제 제기(훅) → 실수 사례 → 정확한 해결법 순차 안내 → 재확인

**전달 기법**
- 스텝 바이 스텝 화면 따라하기 구조
- 비유: "도서관 서적 카테고리 분류 = 재생목록"
- 경고형 사례: "아이들도 내 영상을 볼 수 있지 않나 하시면서 예를 체크하시는 분들"

**리텐션 장치**
- 초반 오픈루프: '이것'이 무엇인지 공개 유보
- 초반 공포: "끝까지 안 보시면 100% 손해입니다"
- 중반 패턴 인터럽트: 예상 외 실수 사례
- 중반 오픈루프: 태그 분석법을 고정 댓글로 넘김 → 채널 락인

## 5. 감정 트리거

주요 감정: 불안 → 안도 → 결핍감

| 감정 | 유발 장치 |
|---|---|
| 불안 | "이것을 끄지 않아서 그럴 것" — 치명적 실수 가능성 |
| 안도 | "딱 3분만, 돈 시간 하나도 안 들어가" — 해결 쉬움 |
| 결핍감 | "태그 넣는 방법은 고정 댓글에 올인원" — 핵심 정보 외부 연결 |

감정 흐름: 불안(훅) → 흥미+긴장(단계별 공개) → 안도(해결책) → 결핍(태그 미공개)
