# 레퍼런스 분석 — 009

- **영상 제목**: 유튜버의 99%가 이 쉬운 10가지 설정을 안해서 조회수 반토막 납니다
- **채널**: 비블 bibl (구독자 4만)
- **조회수**: 85,471 / 좋아요 3,409 / 댓글 256

---

## 1. 주제 요약

초보 유튜버 99%가 놓치고 있는 채널/업로드 세팅 10가지가 조회수를 결정한다.

---

## 2. 썸네일 & 제목 분석

### 썸네일 텍스트 추출

| # | 텍스트 (원문 그대로) | 글자수 | 크기/색상/위치 | 확신도 |
|---|---|---|---|---|
| 1 | `조회수 미친듯이 터지는` | 12자 | 대형 폰트, 흰색, 하단 첫째 줄 | 확실 |
| 2 | `10가지 채널 설정` | 9자 | 최대 폰트 (1보다 큼), 흰색+굵게, 하단 둘째 줄 | 확실 |
| 3 | `OFF` | 3자 | 중간 폰트, 흰색 기울임, 왼쪽 상단 유튜브 로고 아래 | 확실 |
| 4 | `ON` | 2자 | 중간 폰트, 노란색+굵게, 오른쪽 상단 유튜브 로고 아래 | 확실 |

### 썸네일 비주얼 분석

| 항목 | 분석 |
|---|---|
| 구도 패턴 | Before→After (OFF vs ON 대비) — 인물 중심 좌우 분할 |
| 핵심 오브젝트 | 인물 양손 펼침(좌=회색 낮은 막대그래프, 우=청록색 상승 막대그래프) + OFF/ON 유튜브 로고 |
| 색채 | 배경: 어두운 회색/녹색 커튼 톤, 포인트: 청록색(성장 그래프)+노란색(ON), 브랜드컬러 없음 |
| 인물 | 있음, 표정: 진지하고 자신감 있음(카메라 정면 응시), 시선 방향: 정면 |
| 텍스트 공간 | 하단 약 35% 점유, 검정 배경 박스 위 흰색 텍스트 2줄 |
| 시각 장치 | 좌(낮은 회색 막대) vs 우(높이 치솟는 청록 막대) 대비, OFF/ON 라벨로 이분법 강조 |
| 감정 | 박탈감 + 희망 (나는 OFF 상태였구나, 바꾸면 저렇게 된다) |

- 텍스트 레이아웃 요약: 총 4개 텍스트 요소, 계층형 (하단 메인 2줄 + 상단 보조 OFF/ON 2개)

### 제목 분석

- **제목 전문**: 유튜버의 99%가 이 쉬운 10가지 설정을 안해서 조회수 반토막 납니다
- **핵심 장치**:
  - `99%` — 숫자로 "거의 모든 사람이 실수하고 있다"는 박탈감 자극
  - `쉬운` — 진입 장벽 해소 (어렵지 않은데 손해를 보고 있다는 죄책감 유발)
  - `반토막` — 손실 프레이밍 (이익보다 손실이 심리적으로 더 강하게 작동)
  - `안해서` — 행동 부재가 원인임을 명시해 시청자를 잠재적 실수자로 지목

### 썸네일-제목 조합이 만드는 기대감

"이 영상을 클릭하면 내가 지금까지 몰라서 손해 보던 설정 10가지를 알 수 있을 것 같다"

## 3. 훅 & 인트로 분석

### 훅 (첫 3~5초 상당)

> "영상을 열심히 만들고 처음 업로드 하게 되면 생각보다 조회수가 오르지 않아서 대부분 초보 유튜버분들이 아 나는 유튜브에 재능이 없나 보다라면서 쉽게 포기를 합니다."

- **건드린 심리**: 공감형("내 얘기인데?") — 조회수 안 나올 때 느끼는 자기 의심과 포기 충동을 정확히 묘사. 시청자가 "이게 나 얘기잖아"라고 즉각 동일시하게 만든다.
- **효과적인 이유**: 훅이 '잘못된 원인 귀속'을 교정하는 구조다. "재능 없는 게 아니라 세팅 때문"이라는 반전을 예고하며, 시청자에게 희망과 함께 분노(이 정보를 왜 몰랐지)를 동시에 심는다.

### 인트로 (훅 이후 ~ 본문 진입 전)

> "그런데 채널과 업로드 세팅만 바꾸더라도 완전히 조회수는 달라질 수가 있는데요. 지금부터 알려 드릴 세팅들은 제가 24만 21만 7만 채널에 모두 적용해 둔 세팅들이고 지금도 원칙을 두면서 지키고 있는 설정들입니다."

- **역할**: 혜택 + 권위 동시 확인. "완전히 달라질 수 있다"로 결과를 약속하고, "24만·21만·7만 채널"이라는 실적 수치로 신뢰를 즉시 구축한다. 끝까지 봐야 할 이유를 단 두 문장에 압축.
- **평가**: 강한 인트로. 결과(조회수 변화) + 근거(다채널 적용 실적) + 완전 공개 약속("열 가지를 모두 구체적으로")을 3단 구조로 전달.

## 4. 본문 구조 분석

### 전체 구조 요약

1. **1단계: 기본 키워드 설정** — 알고리즘이 채널을 분류하는 원리 + 세 가지 키워드 조사법(크롬 시크릿, 구글 트렌드, 블랙키위)
2. **2단계: 국가/언어 설정** — 노출 언어권 타겟팅
3. **3단계: 중급·고급 인증** — 썸네일 직접 등록 + 15분 이상 업로드 권한 해제
4. **4단계: 썸네일·제목 AB 테스트** — 감이 아닌 데이터로 클릭률 최적화
5. **5단계: 재생 목록 키워드 세팅** — 시청 시간 연장 + 검색 유입
6. **6단계: 아동용 영상 설정 해제** — 댓글·알림·맞춤 광고 제한 방지
7. **7단계: 이메일 등록** — 광고·협업 기회 확보
8. **8단계: 채널 카테고리 설정** — 엉뚱한 시청자 노출 방지 + 단일 주제 운영 원칙
9. **9단계: 비구독자용 트레일러** — 신규 방문자 구독 전환율 제고
10. **10단계: 구독자용 하이라이트** — 기존 구독자 상품/서비스 전환

**정보 배치 순서 패턴**: 문제 제기(왜 필요한가) → 원리 설명 → 실행 방법(클릭 경로 명시) → 효과 확인. 10번 반복하는 단일 템플릿 구조.

### 전달 기법

- **스토리텔링**: 채널 성장 실패 서사로 시작해 단계별 해결 서사로 전환. 추상적 조언이 아니라 본인 운영 채널 사례(24만·21만·7만 채널)로 신뢰성 확보.
- **비유**: "주식 정보 보고 싶은데 갑자기 골프 영상이 떠"처럼 시청자 일상 상황으로 치환해 카테고리 설정 중요성을 설명.
- **데이터/숫자**: `99%`, `24만·21만·7만 채널`, `클릭률 5% 이상`, `키워드 최대 500자`, `1년에 두 계정까지 인증` 등 구체 수치를 매 단계마다 삽입.
- **사례**: 골프 채널 키워드 예시, 본인 컨설팅 서비스 구독자 전환 사례를 직접 인용.

### 리텐션 장치

- **오픈루프 (초반)**: "이 단계를 알고 나면 여러분이 업로드한 영상의 조회수가 확실히 달라지는 걸 바로 느끼실 수 있을 겁니다" — 본문 시작 직전, 결과를 예고하되 내용은 아직 안 줌.
- **전환 멘트 (매 단계)**: "다음 N단계는 ~인데요" 패턴으로 다음 파트를 암시하며 이탈 방지. 10단계 내내 반복.
- **패턴 인터럽트 (8단계 중반)**: "여기서 눈치 채신 분들도 계실 것 같은데요. 유튜브가 채널의 카테고리를 나누어 놨다는 게 힌트되죠." — 시청자를 '눈치 빠른 사람'으로 치켜세우며 주의를 환기.

## 5. 감정 트리거

- **주요 감정**: 박탈감 → 희망 → 실행 의지

| 감정 | 유발 장치 |
|---|---|
| 박탈감 | "99%가 놓치고 있는", "조회수 반토막" — 내가 손해를 보고 있었다는 인식 |
| 희망 | "24만·21만·7만 채널에 모두 적용한 세팅", "딱 한 번만 해 두면 되는 설정" — 나도 할 수 있다는 가능성 |
| 실행 의지 | 각 단계의 클릭 경로 명시("유튜브 스튜디오 → 설정 → ...") — 지금 당장 따라 할 수 있다는 구체성 |

**감정의 흐름**: 박탈감(훅) → 희망+신뢰(인트로) → 단계별 실행 의지(본문) → 안도(마무리: "딱 한 번이면 된다")

## 6. 댓글 반응 분석

### 시청자가 가장 강하게 반응한 포인트

- 댓글 8위 (좋아요 2): "구독자가 6일만에 132가 되었습니다 감사합니다" — 실제 적용 후 결과 공유. 영상 실용성에 대한 직접 검증.
- 댓글 7위 (좋아요 1): "영상대로 세팅했어요. 구독자랑 조회수가 늘었으면 좋겠습니다" — 세팅 적용 후 기대 표현. 시청자가 정보를 받아들이고 실행까지 옮겼음.
- 댓글 4위 (좋아요 0): "좋은 정보라서 보고 또 보고" — 반복 시청. 정보 밀도가 높아 한 번에 소화하기 어렵다는 신호.

**반응 패턴 요약**: "따라 해봤다"거나 "따라 하겠다"는 액션 지향 댓글 다수. 조회수 알고리즘 이론보다 **즉시 적용 가능한 실행 단계**에 반응.

### 시청자 포지션

초보~중급 유튜버. 조회수가 안 나와 좌절 중이거나 막 시작을 준비 중인 사람. 전문 지식보다 "지금 당장 뭘 눌러야 하는지"를 원하는 실행 중심 시청자.

### 우리 영상에서 반드시 다뤄야 할 포인트

- "6일 만에 132명" 같은 **빠른 결과 사례**를 초반에 배치하면 리텐션 효과 기대 가능.
- 반복 시청이 발생한다는 점 → 각 단계를 **체크리스트 형식**으로 명확히 구분해 전달할 것.
- 댓글에 "얼마나 빨리 효과 나오나요?"에 대한 명시적 답변 없음 → 우리 영상에서 **효과 발현 타임라인**을 언급하면 차별화 가능.

## 7. 핵심 약속 역추적

### 제목+썸네일이 약속한 것

"이 영상을 보면 지금 설정 하나 잘못해서 조회수를 절반씩 날리고 있는 내 채널의 문제점 10가지를 구체적으로 알게 될 것 같다"

### 인트로에서 확인해준 것

> "지금부터 알려 드릴 세팅들은 제가 24만 21만 7만 채널에 모두 적용해 둔 세팅들이고 지금도 원칙을 두면서 지키고 있는 설정들입니다."

제목의 약속(10가지 설정)을 인트로에서 그대로 재확인하면서, **신뢰 근거(다채널 적용 실적)**를 추가해 약속을 구체화. 제목보다 신뢰도 한 단계 높임.

### 본문이 실제로 전달한 것

클릭 경로까지 명시된 10단계 실행 체크리스트 — 설정 이유(원리) + 방법(경로) + 기대 효과를 단계별로 패키지화.

### 약속 이행 평가

- **이행 여부**: 이행 (10가지 모두 다룸, 각각 실행 방법까지 제공)
- **오버딜리버리 여부**: 있음 — 제목은 "설정 10가지"만 약속했으나, 본문에서 키워드 조사 3가지 방법(크롬 시크릿·구글 트렌드·블랙키위), AB 테스트 활용법, 채널 카테고리 전략론(단일 주제 운영 원칙)까지 추가 전달. 기대 이상의 분량과 원리 설명 포함.
