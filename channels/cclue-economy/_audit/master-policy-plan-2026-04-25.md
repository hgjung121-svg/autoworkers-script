### 🏆 5순위 — 향후 신규 영상 자동 강제 (시스템)

SSOT v2.0이 자동 강제하므로 사용자 추가 작업 0:
- script-pd 스킬 Lazy Load가 STRATEGY/DRAFT/REVIEW 단계에서 SSOT 자동 로드
- script-writer는 작성 단계에서 14게이트 사전 회피
- script-reviewer는 review.md 끝에 15행 매트릭스(게이트 0 + 1~14) 포함 필수
- verdict 자동: 🔴 1건 = 사용자 confirm / 🟡 1건 = 자동 보정 / 🟠 = 경고

특별 강제:
- **게이트 0 (Retention 40%+)**: 도입부·1분 호기심·9분+ retention 35% 목표

- **EDSA 선언**: 민감 주제(정치·역사·종교) description 첫 단락 표준 템플릿

## 리스크 요약

### 가장 큰 리스크 — Inauthentic Content (게이트 14)
- 한국 = 1위 표적국가 (캡윙 데이터 84.5억뷰)
- 2026-01 16채널 47억뷰 / 연 147억 수익 박탈 사례
- cclue-economy 자동 파이프라인 = 4대 트리거 중 3개 해당
- **변주 강제 룰 적용 안 하면 라이브 30편 + 미업로드 21편 동시 demonetize 위험**

### 두 번째 리스크 — Retention 강등 누적
- 9편 retention 20% 미만
- 조회 1,180·958·660·490 영상 모두 retention 30% 미달 = 다음 영상 부정 신호
- 영상 본문 수정 불가 → **신규 영상에서 회복 시작** (3~6개월 소요)

### 세 번째 리스크 — 한국 12-24 손배법 (2026 중반 시행)
- 5배 징벌적 손배 + 2회 반복 시 10억 과징금
- 단정 표현 + 정치인·기업 명예훼손 결합 시 직격
- 2순위 11편 톤 다운으로 1차 회피

### 네 번째 리스크 — 정알못 NEWS 좌표신고 사례
- 시사 채널 16만 구독자 단일 신고로 삭제
- cclue-economy의 정치 진영 색채 영상 위험
- EDSA 선언 강제 + 객관 톤 유지로 회피

   - 한 번에 일괄 vs 분산? (Inauthentic 회피 위해 분산 권장)

   - 즉시 (오늘 30분~2시간) vs 분할 (며칠에 걸쳐)?
