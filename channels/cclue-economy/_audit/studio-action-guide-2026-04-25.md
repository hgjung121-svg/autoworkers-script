> **점검 보고서**: [uploaded-policy-audit-2026-04-25.md](uploaded-policy-audit-2026-04-25.md)

> **총 예상 시간**: 1시간 (1단계 5분 + 2단계 30분 + 3단계 15분 + 4단계 0분)

## 🔧 사전 준비

1. **Studio 접속**: <https://studio.youtube.com> → 우측 상단 채널 아이콘이 cclue-economy(깊이 읽는 경제)인지 확인
2. **메뉴 진입**: 좌측 사이드바 → **콘텐츠** 클릭 → 동영상 목록 화면 진입
3. **백업**: Studio는 자동 저장이라 별도 백업 불필요. 단, 변경 직후 5분 동안은 다른 작업 자제 (저장 충돌 방지)
4. **시간대 권장**: 트래픽 영향 최소화를 위해 새벽 시간대 권장 (한가할 때)
5. **알림 영향**: description·태그·카테고리 변경은 시청자에게 **알림 안 감**. 제목 변경도 영상 게시 후 24시간 지나면 알림 안 감 — 30편 모두 24시간 경과했으므로 안전.

## 1단계 ─ 28편 일괄 편집 (5분)

### 1-A. AI 디스클레이머 5줄 description 일괄 추가

#### 대상 영상 (28편 — news-0424-evening 한 편만 제외)

저장소 안 22편:
```
0406-morning · 0408-morning · 0409-morning · 0410-morning · 0413-morning
0406-evening · 0407-evening · 0408-evening · 0409-evening · 0410-evening
0413-evening · 0414-evening · 0415-evening · 0416-evening · 0417-evening
0420-evening · 0421-evening · 0422-evening · 0423-evening
weekly-0404 · weekly-0411 · weekly-0418
```

저장소 밖 6편:
```
3-30 (546조 버핏) · 3-31 (정부 민낯) · 4-1 (840조 증발)
4-2 (숨겨진 도미노) · 4-3 (기름값 미쳤다) · 4-5 (코스피 8% 폭등)
```

→ **합계 28편**. news-0424-evening은 이미 모범이라 제외.

#### 작업 절차

1. 콘텐츠 목록에서 28편 체크박스 선택
   - 화면 상단의 "필터" → "게시일" 정렬로 시간 순 보기
   - news-0424-evening만 체크 해제
2. 화면 상단 **편집** 드롭다운 클릭 → **설명** 선택
3. 모드 선택: **"설명 끝에 추가"** (Append to end of description)
4. 텍스트 박스에 아래 5줄 그대로 복붙:

```
⚠️ 본 영상은 AI 음성(TTS)과 AI 보조 도구로 제작되었습니다.
⚠️ 본 영상은 정보 제공 목적이며 특정 종목 매수·매도 추천이 아닙니다.
⚠️ 투자 결정과 결과에 대한 책임은 시청자 본인에게 있습니다.
⚠️ 본 채널은 유사투자자문업 미신고 채널입니다.
⚠️ 데이터 출처는 영상 본문 및 설명 하단 참조.
```

5. **동영상 업데이트** 클릭 → 28편 일괄 적용 (3~5초)
6. 진행 바 100% 확인

### 1-B. 태그 일괄 추가

1. 28편 다시 체크 (또는 같은 선택 유지)
2. 편집 드롭다운 → **태그** → **추가**
3. 입력: `AI콘텐츠, 정보제공, 깊이읽는경제`
4. 동영상 업데이트 클릭

### 1-C. AI 콘텐츠 라벨 (게이트 10 정책 의무)

> ⚠️ Studio 일괄 편집에서 일부 채널은 AI 라벨 항목이 안 보일 수 있음. 그럴 때는 개별 처리.

**일괄 가능한 경우**:
1. 28편 체크 → 편집 드롭다운 → **변경되거나 합성된 콘텐츠** 또는 **Altered or synthetic content**
2. **"AI를 사용하여 음성·영상·이미지 생성됨" 체크**
3. 동영상 업데이트

**일괄 안 보이는 경우 (개별)**:
1. 영상 1편씩 클릭 → 좌측 사이드 메뉴 **"세부정보"**
2. 페이지 하단으로 스크롤 → **"변경되거나 합성된 콘텐츠"** 섹션
3. **"예, 변경되거나 합성된 콘텐츠가 포함되어 있습니다"** 선택
4. **저장**

### ✅ 1단계 완료 검증

샘플 1~2편 클릭 → 세부정보 → description 끝에 5줄 추가됐는지 + 태그 3개 추가됐는지 + AI 라벨 적용됐는지 확인.

## 3단계 ─ 외부 6편 개별 편집 (15분, 편당 2.5분)

> 이 6편은 git에 없으므로 사용자가 영상 본문·기존 description을 직접 봐야 합니다.

### 3-1. 2026-03-30 — "546조원을 현금으로 묻었다 버핏 경고"

- **권장 제목**: `546조원을 현금으로 묻은 버핏의 12분기 매도 행보 — 역사 시그널 해석`
- **수정 포인트**: "역사상 가장 위험한 신호!" 같은 단정 표현 제거. "역사 시그널 해석"으로 톤 다운

### 3-2. 2026-03-31 — "정부가 절대 보여주지 않는 한국 경제의 민낯"

- **권장 제목**: `한국 경제 회복의 두 얼굴 — 수출 역대 최대 vs 가계 체감 경기 분석`
- **수정 포인트**: "절대 보여주지 않는", "민낯" → "두 얼굴", "분석"으로 객관화

### 3-3. 2026-04-01 — "840조가 증발했다. 진짜 범인?"

- **권장 제목**: `3월 한국 증시 840조 시총 감소 — 외국인 매도와 도미노 구조 분석`
- **수정 포인트**: "증발했다", "진짜 범인" → "시총 감소", "구조 분석"

### 3-4. 2026-04-02 — "진짜 위협하는 숨겨진 도미노"

- **권장 제목**: `삼성전자·SK하이닉스에 영향 주는 4월 6일 도미노 — 유가·환율·반도체 연결 구조`
- **수정 포인트**: "진짜 위협하는", "숨겨진" → "영향 주는", "연결 구조"

### 3-5. 2026-04-03 — "기름값이 미쳤다 내 지갑에 벌어질 일"

- **권장 제목**: `유가 62% 폭등 분석 — 4월 6일 이후 한국 가계 부담 시나리오`
- **수정 포인트**: "미쳤다", "내 지갑" → "분석", "가계 부담 시나리오"

### 3-6. 2026-04-05 — "코스피 8% 폭등 다음날 4.5% 폭락 도대체"

- **권장 제목**: `코스피 8% 폭등→4.5% 폭락 — 4월 첫째주 시장 변동성 주간 정리`
- **수정 포인트**: "도대체 무슨 일이 있었나" → "주간 정리"
- **카테고리**: 뉴스/정치 권장

#### 3단계 작업 절차

각 영상마다:
1. 영상 클릭 → 세부정보 → 제목 교체 + description 첫 줄 위에 다음 문장 추가:

```
본 영상은 정보 제공 목적이며 특정 종목 매수·매도 추천이 아닙니다.
```

2. 1단계에서 디스클레이머 5줄 + AI 콘텐츠 라벨은 이미 적용됐으니 추가 작업 없음

## ⚠️ 주의사항

1. **일괄 편집 후 즉시 새로고침 X** — Studio가 백그라운드에서 처리 중일 수 있음. 5분 후 확인.
2. **제목 변경 후 24시간 내 트래픽 변동 모니터링** — 알고리즘이 새 제목으로 재인덱싱하는 동안 노출 일시 감소 가능.
3. **AI 라벨 누락 시 수익화 영향** — 게이트 10 위반은 광고 수익 제한 직격. 1단계 1-C는 반드시 완료.
4. **저장 실패 시** — Studio가 가끔 저장 실패 메시지를 띄움. 다시 저장 클릭하면 됨. 데이터 손실은 없음.
5. **댓글·시청자 반응 변화** — 제목 톤 다운 후 첫 24시간은 클릭률(CTR) 일시 감소 가능. 메모리 [feedback_thumbnail_ab_formula]·[feedback_engagement_questions] 룰로 보완.
