# _inbox — 외부 브리핑 수집 폴더

## 용도
Gemini·GPT·사람 등 외부 소스에서 들어오는 하루 3~4회 브리핑을 날짜별로 쌓아두는 곳.
오후 심층·주간 총정리 트리거가 R0 단계에서 먼저 읽고 주제 후보에 반영.

## 파일 규칙
- 파일명: `YYYY-MM-DD.md` (KST 기준 날짜)
- 하루 여러 브리핑은 **한 파일에 시각별로 섹션 추가**
- 트리거가 실행될 때 그날 파일이 있으면 자동 Read, 없으면 skip

## 파일 템플릿
```markdown
# {YYYY-MM-DD} Gemini 브리핑 모음

## {HH:MM} 1차 브리핑
{브리핑 원문 그대로}

## {HH:MM} 2차 브리핑
{브리핑 원문 그대로}
```

## 수동 저장 방법
사용자가 브리핑을 보내면 Claude가 이 폴더에 자동 저장.
