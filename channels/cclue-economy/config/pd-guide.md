# 경제 대본 작성 가이드

> 벤치마크 채널 분석 기반 (신사임당, 퐁퐁퐁경제학, 경제명탐정, 머니인사이드, 실전경제, 지식브런치, 사건텔러)
> 이 가이드는 매일 자동 생성되는 오전/오후 대본의 품질을 높이기 위한 실행 규칙이다.
> **핵심 원칙: 뉴스를 나열하지 않는다. 뉴스를 해석한다.**

---

## 1. 훅 공식 (첫 10초)

### 검증된 패턴
- **역설적 선언형**: 상식과 반대되는 팩트를 첫 문장에 던진다
  - 예: "석유 한 방울 안 나는 나라한테 제발 팔아 달라고요" (3.3만)
  - 예: "국제질서의 뼈대가 지금 완전히 박살나고 있는 중입니다" (5.7만)
- **따옴표 인용형**: 실제 인물/기관의 발언을 따옴표로 배치
  - 예: "제 2의 스마트폰 탄생한다" (22만)
- **숫자 충격형**: 가장 충격적인 수치 하나를 첫 문장에
  - 예: "항공유 가격이 한 달 만에 95% 올랐어요" (3.3만)
- **연속 부정형**: 시청자가 예상하는 답을 2~3회 연속 부정한 뒤 진짜 답을 제시
  - 예: "이 전쟁을 지휘하는 게 대통령? 아닙니다. 정규군? 그것도 아니에요. 진짜 엔진은 따로 있습니다" (9.2만)

### 실행 규칙
1. 첫 문장에 반드시 **역설, 충격 수치, 또는 직접 인용** 중 하나를 배치한다
2. "오늘은 ~에 대해 알아볼게요" 식의 나열형 도입 절대 금지
3. 훅 직후 "이게 무슨 상황일까요?" 같은 질문을 던져 시청자의 답 찾기 본능을 자극한다
4. 15초 안에 "이 영상을 왜 끝까지 봐야 하는가"가 전달되어야 한다

## 2. 인트로 공식 (훅 후 20~30초)

### 검증된 패턴
- **혜택+공포 동시 투하**: "한국이 세계 최후 공급자(자부심)" + "그 한국의 재고가 4월 말 바닥(공포)"
- **뉴스 무력화 → 깊이 약속**: "뉴스에서 봤죠? 그건 피상적인 단면에 불과해요. 밑바닥까지 해부해 드립니다"
- **도미노 예고**: 영상 전체의 인과 체인을 3줄로 압축 제시

### 실행 규칙
1. 인트로에서 반드시 "이 영상 끝까지 보면 뭘 알게 되는가"를 구체적으로 약속한다
2. 시청자가 이미 아는 정보를 먼저 공감시키고 → "그건 빙산의 일각"으로 뒤집는다
3. 시그니처 표현 "돈의 흐름을 따라가보면"을 인트로에서 1회 사용한다

## 3. 본론 구조 — 레이어 파고들기

### 검증된 구조 패턴
**레이어 파고들기형** (가장 높은 조회수 + 깊이):

```
L1. 표면 뉴스 (What) → L2. 프레임 전환 → L3. 구조적 원인 (Why) → L4. 전략적 심리/역사 → 시나리오 전망
```

**구체적으로**:
1. **L1. 표면 뉴스**: 무슨 일이 벌어졌는가 (팩트+수치) + 생활 번역 ("이게 내 지갑에 어떤 의미?")
2. **L2. 프레임 전환**: "뉴스에서는 여기서 끝이야. 근데 진짜 봐야 할 건 따로 있어" — 시청자의 1차 해석을 뒤집고 더 깊은 레이어로 진입
3. **L3. 구조적 원인**: 왜 이런 일이 벌어졌는가 (지정학적 배경, 경제 구조, 이해관계 충돌)
4. **L4. 전략적 심리/역사**: 각 행위자의 속셈, 과거 전례와 비교, "왜 이렇게 행동할 수밖에 없는가"
5. **전망**: 시나리오 A/B/C + 각 시나리오별 시청자 대응 가이드

### 레이어 전환 패턴 (필수)
- 각 레이어 진입 시 **이전 레이어를 무력화**한다:
  - "뉴스에서는 전부 합의 실패에만 집중해. 근데 진짜 봐야 할 건 따로 있어"
  - "여기까지가 표면이야. 근데 이걸로는 절반만 보는 거야"
  - "원인은 알았어. 근데 왜 이 시점에 이런 결정을 했는지, 이건 역사를 봐야 이해가 돼"
- **최소 2회 레이어 전환**을 본론에 배치한다

### 추가 구조 패턴 (주제에 맞게 선택)

**역추적 분석법** — 공식 명분과 실제 목적이 다른 이슈에서 사용:
- "적이 겨냥한 곳 = 진짜 목적" → 행위자의 행동에서 의도를 역추적
- 예: 이란이 달러 결제 유조선만 봉쇄 → 미국이 지키려는 건 석유가 아니라 달러 패권

**양면 교차 해부** — 내부+외부 요인이 모두 있는 이슈:
- 내부 해부 → "시선을 바깥으로 돌려보면" → 외부 해부 → "이 두 가지를 합치면" 종합
- 단선적 파고들기가 아닌 입체적 깊이를 만든다

**오해 해체 연쇄** — 투자/금융/생활경제 주제에서 효과적:
- 오해 1 제시→해체 → 오해 2 제시→해체 → 모든 저항 제거 후 핵심 논증
- 각 오해 해체가 다음 논점의 전제가 되도록 순서 설계

**불리한 팩트 선제 공개** — 한쪽으로 치우칠 때 신뢰 확보:
- "솔직하게 말하면, [불리한 팩트]야. 이건 숨기고 싶지 않아"
- 그 위에서 "근데 여기서 끝내면 절반만 보는 거야"로 더 깊은 분석 진입

### 실행 규칙
1. 단순 뉴스 나열 절대 금지. 반드시 **인과관계 도미노**로 연결한다 (A→B→C→내 지갑)
2. 모든 파트에 **"글로벌→한국→내 일상" 3단 착지**를 포함한다 (2~3분마다)
3. 긍정 분석 후 반드시 리스크/반론 파트를 넣는다 (균형 필수)
4. 마지막에 "그래서 우리는 어떻게 해야 하는가"로 착지한다
5. 행위자의 주요 행동을 전달할 때, **멈추고 전략적 의도를 해부**한다:
   - "이게 왜 소름 돋는 전략인지 알아?" → 행동→의도→효과 3단 분해

## 4. 비유/숫자 규칙

### 검증된 비유 패턴
- **일상 치환형** (가장 효과적):
  - "식당 주인이 전 세계 손님한테 밥은 해 주고 있는데 정작 본인 쌀이 떨어지고 있는 상황이에요" (3.3만)
  - "호르무즈가 유료 톨게이트가 되는 거예요" (3.3만)
  - "톱니바퀴처럼 맞물려 돌아가는" (5.7만)
- **집단 기억 레퍼런스형**:
  - "제 2의 스마트폰" — 모두가 체감한 과거 변화를 기준점으로 현재 사건의 규모 전달 (22만)

### 비유 프레임 확장법 (3단 확장)
핵심 비유를 1회로 끝내지 않는다. 같은 프레임을 **3단으로 확장**한다:
1. **1단 도입**: "패가 한 장이라도 있는 사람은 서두를 이유가 없어"
2. **2단 구체화**: "그 조커가 뭔지 알아? 호르무즈야"
3. **3단 교훈 연결**: "핵 카드를 내려놓으면 어떻게 되는지 카다피가 보여줬어"

### 프레임 관통 비유 (영상 전체를 하나의 비유로)
핵심 비유를 영상 전체의 **분석 렌즈**로 사용한다:
- 도입: 비유 설정 ("기생수 같은 조직이 국가를 삼켰다")
- 중반: 비유로 현상 해석 ("숙주를 삼키는 과정")
- 클로징: 비유로 결론 회수 ("기생수가 숙주와 함께 파멸해 간다")
- 비유 출처: **신체**(숨통, 혈액, 체온계), **일상**(스위치, 냉장고, 밀키트)에서 가져올 것

### 비유 세계관 재활용 (같은 비유, 다른 논점)
하나의 핵심 비유를 여러 논점에 **재활용**하면 인지 부하가 낮아진다:
- 1차(개념 도입): "아파트 A동은 2억, B동은 1.5억. 차이는 투표권 하나"
- 2차(핵심 차이): "A동 보증금 2억, B동 1.4억. 근데 월세는 같아"
- 3차(복리 효과): "B동 주인은 보증금이 쌌던 덕에 집을 한 채 더 샀다"

### 실행 규칙
1. 추상적 개념이 나올 때마다 즉시 일상 비유를 배치한다
2. 핵심 비유는 1회가 아니라 **프레임으로 유지하며 3단 확장**한다
3. 숫자는 반드시 "맥락 번역"과 함께 제시한다
   - "70.7%" → "기름 10통 중 7통이 중동에서 온다는 거야"
   - "1,400만 배럴" → "우리나라 5일치 사용량"
4. 한 문장에 숫자 2개 이하. 3개 이상 몰리면 문장을 쪼갠다
5. 큰 숫자는 Before→After 대비 쌍으로 제시한다 (70달러→113달러)

## 5. 리텐션 장치

### 검증된 패턴
| 장치 | 예시 | 배치 타이밍 |
|------|------|-----------|
| **전환 멘트** | "근데요. 진짜 무서운 건 유가가 아니에요." | 파트 전환 직전 |
| **오픈루프** | "근데 소름 돋는 게 있어요. 이 한국이요. 지금 엄청 위험합니다." | 중반, 새 긴장 투입 |
| **질문형 인터럽트** | "전 세계가 눈이 한 곳을 향하고 있거든요. 어딜까요?" | 2~3분 간격 |
| **재반전** | "근데 여기서 반전이 있어요. 이 난리 통에 한국 수출이 사상 최대를 찍었어요." | 후반부 |

### 실행 규칙
1. 2~3분 간격으로 리텐션 장치를 배치한다 (빈 구간 없이)
2. 파트 전환 시 반드시 "근데 진짜 문제는 이게 아니야" 류의 전환 멘트를 넣는다
3. 중반에 새로운 변수/반전을 추가해 궁금증을 갱신한다
4. 시그니처 "그런데 여기서 소름끼치는 게" 는 영상당 1회만 사용한다
5. **행동 지시형 참여 장치** (투자/금융 주제에서 특히 효과적):
   - "지금 앱을 켜서 직접 확인해 봐", "본인 계좌의 X와 Y를 비교해 봐"
   - 시청자가 행동하는 순간 영상 정보가 자기 상황과 연결되어 몰입 극대화
   - 초반 1회 + 클로징 1회 배치

## 6. 감정 아크

### 검증된 패턴
```
긴박함/호기심(초반) → 지적 호기심/몰입(중반) → 불안(시나리오) → 통제감/대비감(마무리)
```

### 실행 규칙
1. 공포만 쌓지 않는다. **클로징은 "통제감"으로 착지**한다 — "나도 준비할 수 있다"
2. "이건 남의 나라 일이 아니라 내 지갑 문제"라는 감정 전환을 본론 중반에 배치한다
3. 클로징에서 **역사적 전례를 활용해 통제감을 부여**한다:
   - 예: "걸프전 때도 이랬어. 앞이 전혀 안 보이는 그 시점에서 시장이 바닥을 찍었거든"
   - 불확실성을 "과거에도 이랬고, 결국 이렇게 됐다"로 전환
4. 단일 전망 금지. **최소 2개 시나리오 + 각 시나리오별 대응 가이드**를 제시한다
5. **감정 스케일링**: 같은 감정을 유지하면 마비된다. 스케일을 키워라:
   - 개인(내 투자 손실) → 시장(연쇄 폭락) → 국가(외환위기) → 다시 개인(내 물가/고용)
   - 또는 감정 종류를 전환: 공포 → 지적 호기심 → 통쾌함 → 경각심

## 7. 반면교사 (하지 말아야 할 것)

> 벤치마크 002 영상에서 댓글 상위 2개가 모두 과장 비판이었다. 교훈:

1. **과장 프레이밍 금지**: "전쟁보다 더 끔찍한 게 온다" → 본문에서 귀결 못 하면 신뢰 하락
2. **제목 약속은 반드시 이행**: 제목에서 약속한 것을 본문에서 명확히 답한다
3. **검증 안 된 주장 금지**: 모든 수치와 사실은 검색으로 확인된 것만 사용
4. **감정 과잉 수식 금지**: "충격", "경악", "발칵" 같은 수식어 남발하지 않는다
5. **팩트+데이터가 신뢰의 기반**: "방대한 자료를 팩트로 해설" (좋아요 14)이 가장 강한 긍정 반응

## 8. 자료 수집 3라운드 (뉴스 대본용)

> 오전/오후/주간 뉴스 대본 작성 시 반드시 이 순서로 자료를 수집한다.

### 라운드 1: 속보 팩트 (What — 4회 이상)
- "{TODAY} 미국 증시 마감", "{TODAY} 환율", "{TODAY} 유가", "{TODAY} 경제 뉴스"
- 결과: 오늘 무슨 일이 벌어졌는지 팩트와 수치 확보
- **이 단계에서 핵심 이슈 1개를 선정한다**

### 라운드 2: 배경·원인 (Why — 4회 이상)
- "{핵심이슈} 왜 발생했는지 원인", "{핵심이슈} 역사적 배경", "{핵심이슈} 지정학적 맥락"
- 과거 전례 검색: "{핵심이슈}와 유사한 과거 사건", "과거 {유사사건} 시장 영향"
- 결과: 왜 이런 일이 벌어졌는지 구조적 원인 + 역사적 전례

### 라운드 3: 전망·시나리오 (What's next — 4회 이상)
- "{핵심이슈} 향후 전망 시나리오", "{핵심이슈} 전문가 분석", "{핵심이슈} 한국 경제 영향"
- 결과: 앞으로 어떻게 될 수 있는지 복수 시나리오 + 투자 대응

### 핵심 원칙
- **24시간 이내 기사만 사용** (주간총정리는 해당 주 기사)
- 라운드 1이 끝나면 핵심 이슈를 선정하고, 라운드 2~3에서 그 이슈를 **파고든다**
- 모든 라운드를 거쳐야 "깊이 있는 분석"이 가능하다. 라운드 1만으로 대본을 쓰면 뉴스 나열이 된다

## 9. 역사적 전례 활용

> "깊이"의 핵심은 현재 사건을 역사적 맥락에 놓는 것이다.

### 실행 규칙
1. 핵심 이슈마다 **과거 유사 사건 1개**를 2~3문장 서사로 제시한다
   - 예: "2011년 카다피는 핵을 포기했어. 8년 뒤 정권이 무너졌고, 카다피는 길 위에서 죽었어. 이란이 이걸 모를 리가 없지."
2. 역사 전례는 단순 사실 나열이 아니라 **현재 행위자의 동기를 설명하는 서사**로 쓴다
3. 클로징에서 **역사적 패턴으로 통제감을 부여**한다
   - "걸프전 때도, 이라크전 때도, 가장 불안한 그 시점이 시장 바닥이었어"
4. 과거 사건의 수치도 Before→After 대비로 제시한다
5. **역사 도미노 타임라인**: 과거 전례를 2~3문장이 아니라 **다단계 후속 도미노**까지 그려라
   - 예: 이라크전(2003)→유로 인덱스 폭락→독일/프랑스 0% 성장(5년)→PIIGS 디폴트(2009)→유로 위협 소멸
   - 이 타임라인이 현재 사건의 장기 전개를 예측하는 근거가 된다
6. **원리-역전-법칙 구조**: 보편 원리 선언 → 현재 그 원리가 깨지는 상황 → 역사적 불변 법칙
   - 예: "독재는 공포로 유지돼" → "그 공포가 역전됐어" → "이 단계에 진입한 국가는 살아남은 적 없어"

## 10. 시나리오 실행 가이드

> 단일 전망("오를 것 같다")은 금지. 반드시 복수 시나리오 + 대응을 제시한다.

### 시나리오 구조
각 시나리오에 3가지를 포함:
1. **촉발 조건**: "만약 ~하면" (어떤 상황에서 이 시나리오가 현실이 되는가)
2. **시장 전개**: 유가/환율/증시에 어떤 영향이 있는가
3. **시청자 대응**: "이 경우 우리는 어떻게 해야 하는가"

### 실행 규칙
1. 최소 2개, 가능하면 **3개 시나리오**(최선/기본/최악)를 제시한다
2. 시나리오 후 **핵심 변수 체크리스트 3~5개**를 제시한다
   - "다음 주에 이 3가지만 주시하면 돼: ① IMF 춘계회의 ② 이란 재협상 여부 ③ 환율 1,500원 돌파 여부"
3. 시청자 유형별 가이드를 추가한다 (적극적 투자자 vs 장기 적립식 vs 관망)
