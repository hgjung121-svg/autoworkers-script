# 경제 대본 작성 가이드

> **핵심 원칙: 뉴스를 나열하지 않는다. 뉴스를 해석한다.**

---

## 1. 훅 (첫 5초)
- 첫 문장은 **손실/위기 프레이밍**으로 시작 ("이것 모르면 손해/위험" 구조)
- 첫 5초 안에 **구체적 숫자 최소 1개** 투하 ("금리 0.25% 올랐을 뿐인데, 연이자 60만 원이 더 나간다")
- 시청자 **내면 독백을 대신 말한다** — 질문형("알고 있으세요?")보다 독백형("그래서 뭐가 달라지는데? 다들 이렇게 생각하잖아")
- "오늘은 ~에 대해" 식 나열형 도입 금지

## 2. 인트로 (훅 후 20~30초)
- "끝까지 보면 뭘 알게 되는가" 구체적 약속
- **신뢰 근거 즉시 제시** ("한국은행 보고서 36페이지를 직접 뜯어봤는데", "이번 주 증권사 리포트를 5개 비교해봤더니")
- 시청자가 아는 정보 공감 → "그건 빙산의 일각"으로 뒤집기
- 시그니처 "돈의 흐름을 따라가보면" 1회 사용

## 3. CTA (인트로 직후!)
- **훅+인트로 끝나고 본론 진입 전에 배치** (클로징이 아닌 초반!)
- "구독 좋아요 알림설정" 직접 나열 금지 — 영상 주제와 연결해 자연스럽게
- 예: "이런 분석 놓치기 싫으면 구독 한번 눌러두고 시작하자"
- 클로징에는 채널 재방문/다음 영상 예고만

## 4. 본론 구조
### 레이어 파고들기 (필수)
```
L1. 표면 뉴스(What) + 생활 번역 → L2. 프레임 전환("진짜 봐야 할 건 따로 있어") → L3. 구조적 원인(Why) → L4. 전략적 심리/역사 → 시나리오 전망
```
- **최소 2회 레이어 전환** (이전 레이어를 무력화하며 진입)
- 인과관계 도미노 (A→B→C→내 지갑)
- 모든 파트에 **글로벌→한국→내 일상** 3단 착지 (2~3분마다)
- 행위자 행동 → **의도 해부** (행동→의도→효과)

### 역설/반전 프레이밍
- 파트 도입마다 상식을 뒤집는 한 문장 배치 ("안전자산이 오히려 위험하다", "금리 인하가 악재")
- 근거 제시 후 "뉴스에서 안 알려주는 / 보고서를 직접 뜯어봤더니" 정보 희소성 프레이밍

### 추가 구조 (주제에 맞게 선택)
- 역추적 분석법 / 양면 교차 해부 / 오해 해체 연쇄 / 불리한 팩트 선제 공개

## 5. 비유/숫자
- 추상 개념 등장 **즉시** 1초 비유 배치 (설명 후 비유 X, 비유가 설명을 대체)
- 감정이 실린 비유 ("양적완화 = 불난 집에 기름 붓는 격")
- 숫자는 반드시 **Before→After 대비 쌍** ("금리 1.75%→3.5%, 월이자 42만→84만 원")
- 한 문장에 숫자 2개 이하

## 6. 리텐션
- 2~3분 간격으로 배치 (전환 멘트/오픈루프/질문형 인터럽트/재반전)
- 파트 전환 직전에 **오픈루프 설치** ("여기까지가 1차 충격이야. 근데 진짜 문제는 다음이거든")
- "그런데 여기서 소름끼치는 게" 영상당 1회만

## 7. 감정 아크
```
공포/박탈감(초반 30%) → 공포-안도 반복 사이클(중반 40%) → 안도+실행 의지(후반 30%)
```
- **파트마다 공포→안도 사이클 반복** (한 이슈 해결 후 새로운 위기 시작 = 롤러코스터)
- 공포만 쌓지 않는다. **클로징은 통제감+실행 가능한 행동**으로 착지
- 감정 스케일링: 개인→시장→국가→다시 개인
- 역사적 전례로 통제감 부여
- 최소 2개 시나리오 + 대응 가이드 + 핵심 변수 체크리스트 3~5개

## 8. 금지사항
1. 과장 프레이밍 금지 (본문에서 귀결 못 하면 신뢰 하락)
2. 제목 약속 반드시 이행
3. 검증 안 된 수치/주장 금지
4. 감정 수식어 남발 금지 ("충격", "경악", "발칵")
5. 뉴스 단순 나열 금지

## 9. 자료 수집 3라운드
1. **라운드 1 (What, 4회+)**: 속보 팩트·수치 → 핵심 이슈 1개 선정
2. **라운드 2 (Why, 4회+)**: 배경·원인·역사적 전례 + **기사 원문 2~3개 WebFetch**
3. **라운드 3 (What's next, 4회+)**: 전망·시나리오·한국 영향
- 24시간 이내 기사만 사용

## 10. 제목/썸네일 설계
- **제목**: 손실 프레이밍 동사 + 숫자 최소 1개 + 정보 희소성('진짜 이유'/'아무도 모르는')
- **썸네일**: Before→After 수치 대비 + 경고형 문구 2줄(7~12자) + 어두운 배경+빨강/노랑 포인트
- **역할 분담**: 썸네일=수치 충격(시각), 제목=손실 경고(텍스트). 같은 정보 반복 금지
