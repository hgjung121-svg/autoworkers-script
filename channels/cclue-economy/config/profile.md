# 깊이 읽는 경제

## 장르
경제/시사

## 톤 & 말투
- 스타일: 밤에 친구한테 이야기 들려주듯
- 상세: 나직하고 편안한 말투. 반말 베이스로 자연스럽게 이야기하되, 핵심 데이터나 결론 강조 시 톤을 살짝 올림. 과장 없이 담담하게, 그러나 긴장감은 유지.

## 금지 표현
- "오늘은"
- "구독 좋아요 알림설정" (직접적 CTA 금지 — 아래 CTA 규칙 참조)

## CTA (구독/좋아요/알림 유도) 규칙
**훅+인트로 직후, 본론 진입 전에 배치한다** (클로징이 아닌 초반!). 직접적으로 "구독 좋아요 알림설정 눌러주세요" 하지 않고, 영상 주제와 연결해서 자연스럽게 유도한다.
- 초반 CTA 예: "이런 분석 놓치기 싫으면 구독 한번 눌러두고 시작하자"
- 초반 CTA 예: "이 흐름, 매일 새벽에 알려줄게. 알림 설정 해두면 놓치지 않아"
- 클로징에는 구독 CTA 대신 **다음 영상 예고/채널 재방문 유도**만 배치

## 시그니처 표현
- "돈의 흐름을 따라가보면" — 주제 전환 또는 핵심 분석 진입 시 사용

## 서사 구조
- 선호: 몰락 서사, 반전 서사
- 기피: 나열형 (이유: 경제 주제는 인과관계가 핵심이므로 단순 나열은 설득력이 떨어짐)

## 관점
항상 소비자/일반인 관점에서 이야기한다. "이게 우리 생활에 어떤 영향을 주는가?"를 기준으로 경제 현상을 해석하고, 전문 용어는 일상 언어로 번역해서 전달한다.

## 정보 전달 방식
숫자/데이터 중심. 주장의 근거를 구체적인 수치로 뒷받침하되, 숫자를 나열하지 않고 맥락과 함께 제시한다. "매출이 43% 하락했다"처럼 팩트를 먼저 던지고, 그 의미를 풀어주는 방식.

## 타겟 시청자
**1차 타겟**: 경제에 관심은 있지만 전문 지식은 부족한 20~40대 직장인. 뉴스는 보지만 깊이 파고들 시간이 없는 사람들. "그래서 이게 나한테 무슨 의미인데?"에 답해주는 콘텐츠를 원하는 시청자.

**확장 타겟**: 50~70대 시니어 시청자도 함께 진입 (실제 댓글 분포에 따라 — 메모리 [project_viewer_content_requests]). 메모리 [feedback_retention_general_viewer]에 따라 누구든 끝까지 볼 수 있게 설계. 전문용어 즉시 생활 번역 의무.

## 검색 키워드 사전 (audience_keywords)

영상별로 다음 카테고리에서 **2~3개 이상 키워드를 제목·설명·태그에 자연스럽게 분산**한다. 첫 100자 SEO 룰 ([feedback_description_first_100_chars]) 적용 시 이 사전 참조.

| 카테고리 | 핵심 검색 키워드 |
|---|---|
| **돈·자산** | 금리·대출·예금·적금·환율·달러·엔화·주식·배당·ETF·ISA |
| **부동산** | 집값·전세·월세·아파트·재건축·종부세·양도세·1주택자·다주택자 |
| **노후·연금** | 국민연금·퇴직연금·노후 준비·노후 자금·은퇴·연금저축·IRP |
| **정책** | 정부 발표·한국은행·기재부·금융위·금감원·국토부·세금·세제 개편 |
| **생활** | 물가·전기세·가스비·의료비·건강보험·실손보험·생활비 |
| **위기·기회** | 위기·폭락·폭등·충격·비상·기회·호재·기준금리·인플레이션 |
| **시니어 한정** | 시니어 재테크·60대 투자·노후 대비·은퇴 후 |
| **세대 한정** | 30대 재테크·40대 자산관리·MZ세대 투자 |

### 키워드 선정 우선순위
1. 영상 핵심 주제 카테고리에서 2개
2. 영상 페인포인트 카테고리에서 1~2개
3. 채널 브랜딩 (밤·돈의 흐름·심층 등) 0~1개

### 트렌딩 키워드 추가 룰
- 영상 제작일 기준 Google Trends / Naver Datalab 1주 핫키워드 1개 자연스럽게 포함
- 단, 영상 주제와 자연스럽게 연결될 때만
