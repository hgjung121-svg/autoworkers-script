## 체크리스트 결과

### 1. 핵심약속 검수

- [x] **Hook에서 핵심약속을 바로 이행하고 있는가**: 통과. "전부 다 좋아지고 있어. 근데 이상하지 않아? 왜 아무도 좋아졌다고 안 느끼는 거야." — 경제 지표와 체감 사이의 괴리를 즉시 제기하며 핵심약속("찝찝함의 정체를 숫자로 확인")을 바로 이행한다.
- [x] **각 파트가 핵심약속 이행에 기여하는가**: 통과. 파트 1~4 모두 "공식 숫자 인정 → 숨겨진 절반 폭로 → 내 통장에 미치는 영향" 구조를 반복하며 핵심약속을 이행한다. 클로징에서 "지금 당장 할 수 있는 4가지" 행동 지침까지 제공하여 "내 돈을 지키는 구체적 행동 지침" 약속도 완결한다.
- [x] **마무리에서 핵심약속이 완결되는가**: 통과. "돈의 흐름을 따라가보면 결국 준비한 사람이 살아남아"로 시그니처를 활용하며 핵심약속을 완결한다.

### 2. 구조 검수

- [x] **outline.md 구조대로 작성되었는가**: 통과. Hook & Intro → 파트 1(수출) → 파트 2(고용) → 파트 3(물가) → 파트 4(재정/부채) → 클로징(4가지 행동) 순서가 outline.md와 정확히 일치한다.
- [x] **확정 제목이 암시하는 구조를 따르는가**: 통과. "정부가 절대 보여주지 않는 한국 경제의 민낯" — 4개 파트에서 각각 공식 발표의 이면을 폭로하는 구조가 제목과 부합한다.
- [x] **타겟 러닝타임 범위 안에 들어가는가**: 통과. 대본 본문 약 5,916자(헤더 제외, 띄어쓰기 포함). 타겟 ~6,000자(15분) 대비 99%. 예상 러닝타임 약 14.8분으로 허용 범위 안이다.
- [x] **Hook과 Intro가 hook-intro.md와 일치하는가**: 통과. draft.md의 Hook & Intro가 hook-intro.md 원문과 글자 단위로 동일하다.
- [x] **각 파트의 감정 흐름이 outline.md 설계와 맞는가**: 통과. P1(납득→불안), P2(분노 누적), P3(배신감→경각심), P4(경각심→불안 최고조), 클로징(안도→행동 의지→자부심) 모두 outline의 감정 설계와 일치한다.
- [x] **리텐션 장치가 파트 전환 지점에 배치되었는가**: 통과. 아래 오픈루프/전환멘트가 모두 배치됨:
  - P1→P2: "수출만 그런 게 아니야. 고용 숫자도 똑같은 짓을 하고 있어." (outline과 일치)
  - P2→P3: "그러면 물가는? 2%래. 진짜일까." (outline과 일치)

  - P4→클로징: "자 여기까지가 숨겨진 절반이야. 근데 이 영상을 그냥 무서운 이야기로 끝내고 싶지 않아." (outline과 일치)
- [x] **verified-data.md의 검증된 데이터만 사용했는가**: 통과. 아래 신규 주장 검증 섹션 참조. 1건의 산술 파생 주장이 있으나 검증됨.

### 3. 문체 검수

- [x] **쉼표 남발 여부**: 통과. 문장이 대체로 짧고 마침표로 끊겨 있다. 쉼표 남발 없음.
- [x] **따옴표 남발 여부**: 통과. 따옴표 사용이 비유나 강조에 한정되어 있고 과도하지 않다.
- [x] **쉬운 말 사용 여부**: 통과. 전문 용어를 사용할 때마다 즉시 풀어서 설명한다. (예: "근원물가라는 게 있어. 식료품이랑 에너지를 빼고 측정하는 물가야.", "적자성 채무라고 해. 도로를 깔거나 공장을 짓는 투자용 빚이 아니라 쓰면 사라지는 돈을 빌리고 있어.")
- [x] **용어 순화 적용 여부**: 통과. "USTR 301조 조사"처럼 불가피한 고유명사 외에는 일상 언어로 순화되어 있다.
- [x] **숫자 과밀 여부**: 통과. 대부분의 문장에서 숫자가 1~2개로 제한되어 있고, 숫자를 제시한 뒤 비유나 맥락 설명이 바로 따른다. 한 문장에 숫자 3개 이상 몰린 경우가 없다.
- [x] **장황한 반복/불필요한 부연 여부**: 통과. 각 파트가 타겟 분량(~1,200자) 범위 내이며 동일 메시지 반복이 없다.

### 4. 톤 검수

- [x] **채널 프로필 말투/톤 준수 여부**: 통과. 반말 베이스, 나직하고 담담한 어조, 핵심 데이터 강조 시 톤 상승. "돈의 흐름을 따라가보면" 시그니처 활용(인트로, 파트 1, 클로징에서 사용). "오늘은" 금지 표현 미사용("오늘"은 사용하나 hook-intro.md에서 이미 확정된 문구).
- [x] **outline.md 프로젝트별 톤 준수 여부**: 통과. outline.md에 정의된 "반말 베이스, 나직하고 담담하게 — 긴장감은 유지" 톤이 전체적으로 일관된다.

## 총평

대본이 outline.md의 구조, 감정 설계, 리텐션 장치를 충실히 반영하고 있다. 모든 사실적 주장이 verified-data.md에 기반하며, 유일한 산술 파생 주장("연간 이자 4조원")도 WebSearch로 검증 통과했다. 채널 프로필의 톤과 문체 규칙을 준수하고, 러닝타임도 타겟 범위 안에 정확히 들어온다. 각 파트에서 "공식 발표 인정 → 숨겨진 절반 폭로 → 내 통장에 미치는 영향" 반복 구조가 일관되게 작동하며, 클로징의 4가지 행동 지침이 핵심약속을 완결한다.
